# 多数元素

[](https://leetcode.com/problems/majority-element/) [## 多数元素- LeetCode

### 给定大小为 n 的数组 nums，返回多数元素。多数元素是出现次数超过…

leetcode.com](https://leetcode.com/problems/majority-element/) 

**见解**:

*   自然，简单的解决方案是用一个计数器告诉我们每个元素的幻影数，然后返回最大值
*   然而，我们将使用 **O(1)内存，**意味着无论我们有多少个元素，我们都将使用*相同数量的内存*
*   想法很简单:我们遍历数组，每一步都有一个**多数竞争者**(可能是真的也可能不是)。
*   如果我们遇到它作为当前元素，我们**将计数增加 1** 。否则，我们**将其减少 1** 。
*   如果计数器达到零，候选元素可能不是多数元素，因此当前项目成为新的候选元素
*   如果多数元素**保证存在**，那么它保证出现足够的次数，这样最终的多数候选人将是真正的
*   如果**多数元素可能不存在**呢？在这种情况下，我们要做的就是再次遍历数组，这次只计算它的幻影数。如果这个数足够大，我们就声明它是一个多数元素
