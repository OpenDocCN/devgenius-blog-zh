属性，也称为类成员变量、属性和字段，是类的特征。想象它们是放在类的花括号内的变量，而不是放在方法本身内。

属性可以直接初始化，也可以在通过构造函数实例化对象时初始化；我们稍后将讨论构造函数。

基于前面的关键字，属性可以具有不同的可见性:

  • 公开
  • 私人
  • 受保护的

后面还将探讨可见性，但目前私有属性仅对对象可见，公共属性在对象内部和外部可见，受保护的属性对对象及其子对象可见。

狗类

可视化属性的最简单方法之一是通过观察动物的特征。比如狗有特定的眼睛颜色、身高、体重等；这条狗会脱毛；狗可以是特定的品种，有特定的皮毛颜色。为了形象化这一点，我们可以创建一个 Dog 类，并向它添加一些属性。为了简单起见，我们将保持所有属性公开。

一旦我们实例化了我们的 Dog 类，我们就可以通过询问对象的毛发颜色来引用这些属性。

每当我们实例化 Dog 类时，我们都有它附带的默认特征。你已经可以看到为什么我们需要构造函数的伏笔:提示，不是每只狗都有黑色和褐色的皮毛。但是，这是我们的计划，我们可以随心所欲地使用它。我们甚至可以改变这些特征。在上面的例子中，我们将狗的皮毛颜色从黑色和棕褐色改为黑色。当我们打印出狗的皮毛颜色时，它是黑色的。

我已经删除了上面的大部分对象属性，留下了一个，燃料类型，因为 PHP 有一些有趣的行为。如果我们修改燃料类型并给它分配一个数字字符串，它仍然可以工作。例如:

$兰博基尼->fuel _ type = " 89 "；

如果你再次 var_dump 这个对象，你仍然会得到完全相同的结果。PHP 会自动将数字字符串转换为 int 类型。你不能对非数字字符串做同样的事情。

$兰博基尼->fuel _ type = " Dino "；
