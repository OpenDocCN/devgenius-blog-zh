# 怎么打电话

## 饼干

Cookies 放在*文档*全局对象中。尽管它将被解析为键值样式的对象，但是没有友好的 API 来作为对象进行读/写。开发人员需要手动编写字符串。

![](img/01a74cdd406084835a309dbb4907d217.png)

## 会话存储和本地存储

两者都支持简单的读/写接口，但是类似于 JSON，但是键和值都必须是字符串。
