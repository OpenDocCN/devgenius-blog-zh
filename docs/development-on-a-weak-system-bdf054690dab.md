# 通常，我们可以从大多数中等功能的系统进行开发，但事实并非如此，为什么呢？

如果只是打开一个代码编辑器，一个开发服务器，任何能满足你的情况的东西，那就很好了，但是在今天的世界里，我们也有更多的辅助应用程序，这就成为了系统滞后的原因。

# 现在要做什么-(步骤)

## 1)尽可能使用最少的 RAM

## —如果可能，增加 RAM(内存)

增加它会给你空间来并行打开更多的应用

## —关闭所有应用程序，尤其是目前基于电子的应用程序，寻找它们的替代浏览器

> “对于那些不了解电子应用程序的人来说 **-(简单地说)**这些应用程序的用户界面是通过网络技术构建的。从逻辑上讲，这些应用程序仍然可以在浏览器上运行，相比之下，就像 how react-native 一样

Skype—[https://web.skype.com/](https://web.skype.com/)

## 2)转向 sublime 等编辑器，因为它们比 VSCode、ATOM 等优化得多，vs code、ATOM 等基于电子构建，但功能不丰富。

现在，它占据这么多内存(至少 200 兆字节或更多——实事求是地说)实际上有很好的理由，因为它的设计方式与同类产品相比，如果你对它感兴趣，可以问谷歌先生。

它的主要目的是提供一个真正好的 UI(作为一个开源软件，不使用 web 技术实现这样的视觉效果是不可能的)和简单的扩展添加。

## 3)直接离开那家公司，他们甚至不会为你提供完成工作所需的最低工资。

## 4)如果你连“*第 3 点”*都做不到，那么我真的很抱歉大声说出来，你不够尊重你的工作，以至于无法区分工作和滥用职权。

如果你不能容忍我的这种说法，那么我很抱歉。作为一个人，由于我的家庭起义，我不会容忍工作虐待。
