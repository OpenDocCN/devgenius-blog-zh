函数可以返回值:我说过了。当我们之前看函数的时候，那些函数只是*重复*一些字符串。尽管这是函数的一种用途，但函数比简单的*回显* out 语句更强大。我们可以使用函数做一些事情，然后*返回*一个计算出来的值给我们。然后，该值可以被*回显*或赋给一个变量。

要返回一个值，您将在函数内部使用 *return* 关键字。确保 *return* 语句是函数中的最后一条语句，因为 PHP 将在执行 *return* 语句后退出函数。

1.  第 11 行调用了函数 *assign_one()* 。
2.  PHP 进入函数 *assign_one()* ，将值 1 赋给一个局部变量 *$x* 。

如你所见，PHP 永远不会到达 *return* 语句之后的语句。

函数可以返回任何数据类型:它们不需要返回整数值。您也不必像我们到目前为止所做的那样，将函数调用赋给一个变量或在函数内部使用一个局部变量(我这样做只是为了明确)。

在下面的例子中，函数 *display_two()* 返回字符串 *two* 。我们可以在 *echo* 语句中连接函数调用。这样，PHP 看到了 *echo* 语句并开始执行。它尝试将两个值合并在一起，因为它遇到了串联运算符。在这样做之前，它首先必须调用函数 *display_two()* ，该函数返回值 *two。*两个字符串被连接，完整的字符串被*回显*出来。

让我们看看返回一个额外的数据类型:数组。

这一次，数组返回时没有先赋给变量。我们可以把这个函数赋给一个变量，这个变量会把数组作为它的值。在这个例子中，我们只是将返回值传递给 *var_dump()* 函数，这样它就可以显示在屏幕上了。
