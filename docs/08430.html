例如，如果要访问类外的常数，可以在实例化的对象，甚至是类名后添加双冒号操作符。如果您正在访问类内部的常量或被覆盖的属性或方法，您可以使用 self 、 parent 或 static 关键字。我们将在本文中讨论自身和父；我们将在下一篇文章中讨论静态。

Self 用于访问类自身内部的常量，而 parent 用于访问继承的常量，或者被覆盖的属性或方法。
