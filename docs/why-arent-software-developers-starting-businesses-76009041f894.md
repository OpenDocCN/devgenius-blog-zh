# 软件开发者为什么不创业？

> 原文：<https://blog.devgenius.io/why-arent-software-developers-starting-businesses-76009041f894?source=collection_archive---------2----------------------->

![](img/2b7c548dd81cce44492948f0d2be17a2.png)

你有没有想过为什么软件开发人员不自己创业？这个问题在我脑海里萦绕了很久。毕竟，如今有如此多的应用程序可以从有人负责开发和营销过程中受益。问题在于，尽管开发者拥有开发成功应用所必需的技能和知识，但大多数人都不想从头开始创建一家公司所涉及的所有工作——尤其是当他们可以通过做其他事情赚更多钱的时候。

软件开发人员在应用程序世界中处于独一无二的赚钱位置。但是他们为什么不创业呢？如果你是一名软件开发人员，并且一直想知道为什么你的很多朋友没有自己创业，这篇文章将帮助你解释其中的一些原因。

# 开发一个应用程序很容易。围绕该应用开发成功的业务更加困难。

如果你正在开发一个应用程序，这很容易。你写代码，上传文件，然后等着别人下载。就是这样——就是这样！

但如果你想围绕这个应用建立一个成功的业务，那么事情就会变得复杂得多，因为现在我们必须处理营销(创建引人注目的内容，以便人们分享我们的帖子)、销售策略和客户支持问题(当有人在购买时遇到问题时会发生什么)。这些只是几个例子；建立一个基于应用的企业，不仅仅是简单地写代码，还涉及到更多的过程。

# **为什么软件开发人员想要创业？**

软件开发商可以赚很多钱。如果你是一名软件开发人员，有很多方法可以从你的技能和才能中赚钱。你可以开自己的公司，或者为别人的公司做自由职业者。或者也许你已经在这个行业有了一份工作，这允许你的时间表和收入有一定的灵活性——这取决于你想投入多少时间和精力！

# **应用开发者没有时间创业。**

开发者没时间创业。他们已经很忙了，而且专注于日常工作。

软件开发人员对创业或风险不感兴趣，因为这会让他们偏离他们最擅长的事情:为其他需要(或想要)软件的人开发软件。

# 你没有市场知识。

你需要知道你的客户想要什么，如何与其他应用竞争，如何营销你的应用并产生收入。如果你没有这些东西，那么你就不太可能作为一名软件开发人员成功创业。

很容易听到你的应用应该“解决一个问题”找到那个问题是困难的部分。通常，这开始于你在一个特定的领域工作，经历了对特定软件的挫败感，然后你自己说，“要是我能想到这个就好了。”

# 不能保证它一定有效。

不保证您的应用程序能够正常工作。营销很难，很难让用户下载你的 app(或者付费)，回来再用，或者推荐给别人。如果你自己不是一个好的营销者，那么让别人参与营销可能比你自己开始更难。

# **结论**

了解软件开发人员在应用程序世界中的独特地位以及如何利用这一优势至关重要。不幸的是，恐惧和计划是阻止软件开发人员创业的原因，就像其他人一样。
