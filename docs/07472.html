React 路由器 是 React 中用于路由的库。它帮助我们浏览一个 React 应用程序中的各个组件。创建一个小应用程序将帮助你练习和理解如何开始使用 React 路由器 。因此，让我们创建一个小的 React 应用程序。我们将使用 React 路由器 在这些组件之间导航。
