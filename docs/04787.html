使用 React 套件库开始 React 开发—选择下拉菜单

原文：https://blog.devgenius.io/developing-vue-apps-with-the-quasar-library-select-dropdown-aa3d391a2128?source=collection_archive---------7-----------------------#2021-05-06

React Suite 是一个有用的 UI 库，让我们可以轻松地将许多组件添加到 React 应用程序中。

在本文中，我们将了解如何使用它向 React 应用程序添加组件。

选择下拉菜单

我们可以用SelectPicker组件在 React 应用程序中添加一个选择下拉菜单。

例如，我们可以写:

import React from "react";
import { SelectPicker } from "rsuite";
import "rsuite/dist/styles/rsuite-default.css";const data = [
  {
    label: "Apple",
    value: "apple",
    role: "Fruit"
  },
  {
    label: "Orange",
    value: "orange",
    role: "Fruit"
  },
  {
    label: "Lettuce",
    value: "lettuce",
    role: "Vegetable"
  }
];export default function App() {
  return (
    <div className="App">
      <SelectPicker data={data} />
    </div>
  );
}

添加一个简单的下拉列表。

label显示给用户。

当我们选择一个项目时，value被用作选择值。
