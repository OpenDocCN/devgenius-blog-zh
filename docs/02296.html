第 27 天—递增三联体序列🏁

目的🏹

给定一个未排序的数组，返回数组中是否存在长度为 3 的递增子序列。

从形式上看，该职能应:

Return true if there exists i, j, k
such that arr[i] < arr[j] < arr[k] given 0 ≤ i < j < k ≤ n-1 else return false.

注:你的算法要在 O( n )时间复杂度和 O( 1 )空间复杂度下运行。

Example🕶

Input: [1,2,3,4,5]
Output: trueInput: [5,4,3,2,1]
Output: false

关注 代码之家 s 了解编程面试世界的最新动态。

密码👇

边缘案例

  1. 因为我们必须比较三个元素，如果给定的数组长度小于三。返回假🔚

算法👨‍🎓

  1. 在这个问题中，我们必须找到三个递增顺序的元素。
  2. 将三个变量初始化为整数最大值。
  3. 对于每个索引，检查当前数字是否小于这些变量中的任何一个。如果它小于它，将变量更改为指向当前元素。
  4. 重复这个过程，直到数组结束，如果这三个变量已经从它们的初始值改变，那么数组中存在一个递增的顺序。
  5. 否则返回 False。🔚

复杂性分析

时间复杂度:O(N) 因为我们只遍历一次。
空间复杂度:O(1) 由于我们只使用了三个附加变量。
