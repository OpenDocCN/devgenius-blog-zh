使用 AND / OR 运算符

使用OR可以显示包含搜索词或关键词的搜索结果。也就是说，我们正在搜索 X 或 Y。这将返回包含 X 或 y 的搜索结果

而另一方面，AND搜索两个搜索词或关键词的存在，即搜索 X 和 Y 将返回包含 X 和 Y 的搜索结果

按文件类型搜索

我们可以按照以下语法从 google 获得基于特定文件类型的搜索结果:filetype:TypeOfFile SearchTerms

当你寻找某个文件时，这是很方便的。Google 倾向于返回基于所提供的文件类型的结果。
