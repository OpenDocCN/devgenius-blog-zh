## 自定义库

库可以被认为是一组相关的函数、变量和代码。

要在 C 中创建一个库，我们需要以下组件:

*   源文件
    — source.c
    —实现
*   头文件
    — header.h
    —原型清单
*   打电话给
    ——我们如何与图书馆互动
