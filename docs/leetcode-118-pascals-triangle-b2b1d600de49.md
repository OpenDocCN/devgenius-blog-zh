# 问题:>

给定一个整数`numRows`，返回**帕斯卡三角形**的第一个数字。

在**帕斯卡三角形**中，每个数字都是其正上方两个数字的和，如图所示:

![](img/c7fdb0f75d6f369f2e47a231a807ac92.png)

**实施例 1:**

```
**Input:** numRows = 5
**Output:** [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
```

**实施例 2:**

```
**Input:** numRows = 1
**Output:** [[1]]
```

**约束:**

*   `1 <= numRows <= 30`
