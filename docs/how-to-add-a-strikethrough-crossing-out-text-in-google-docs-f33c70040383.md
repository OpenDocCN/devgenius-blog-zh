换句话说，这是一种表示反对的方式，或者简单地说，以前写的东西现在不适用了。

在谷歌文档中，我发现删除线选项并不总是很容易找到。它不在大多数平台的顶部工具栏中(我没有检查所有的平台，但我确实发现它经常不在那里)，它也不是很多人都知道的像`command + C`或`control + C`这样复制文本的快捷方式。

# 那么我们如何在谷歌文档中删除文本呢？

这取决于你用的是 MacOS 还是 Windows。

## 苹果个人计算机

如果您使用的是 MacOS(大多数情况下也就是使用苹果电脑)，请突出显示您想要删除的文本，然后点击以下键盘快捷键:

```
command + shift + X
```

## Windows 操作系统

使用 Windows？然后，您需要突出显示想要删除线的文本，然后点击以下键盘快捷键:

```
alt + shift + 5
```

## Mac 或 Windows，没有键盘快捷键

如果你不喜欢使用键盘快捷键，那么你可以手动点击删除文本。

为此，进入顶部工具栏的**格式**。

然后，将鼠标悬停在**文本**上。最后，你可以点击**删除线**。您还会看到与您所在的操作系统相对应的键盘快捷键。
