使用Quasar库开发Vue应用程序—徽章

原文：https://blog.devgenius.io/developing-vue-apps-with-the-quasar-library-badges-2643bd2cb762?source=collection_archive---------6-----------------------#2021-01-15

Quasar是一个流行的Vue UI库，用于开发好看的Vue应用程序。

在本文中，我们将了解如何使用Quasar UI库创建Vue应用程序。

徽章

我们可以添加一个带有q-badge组件的徽章。

例如，我们可以写:

<!DOCTYPE html>
<html>
  <head>
    <link
      href="https://fonts.googleapis.com/css?family=Roboto:100,300,400,500,700,900|Material+Icons"
      rel="stylesheet"
      type="text/css"
    />
    <link
      href="https://cdn.jsdelivr.net/npm/quasar@1.12.13/dist/quasar.min.css"
      rel="stylesheet"
      type="text/css"
    />
  </head>  <body class="body--dark">
    <script src="https://cdn.jsdelivr.net/npm/vue@^2.0.0/dist/vue.min.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/quasar@1.12.13/dist/quasar.umd.min.js"></script>
    <div id="q-app">
      <q-layout
        view="lHh Lpr lFf"
        container
        style="height: 100vh;"
        class="shadow-2 rounded-borders"
      >
        <q-badge color="blue">
          badge
        </q-badge>
      </q-layout>
    </div>    <script>
      new Vue({
        el: "#q-app",
        data: {}
      });
    </script>
  </body>
</html>

我们添加q-badge组件来添加一个徽章。

color是徽章的背景色。

我们可以用text-color道具改变文本颜色:
