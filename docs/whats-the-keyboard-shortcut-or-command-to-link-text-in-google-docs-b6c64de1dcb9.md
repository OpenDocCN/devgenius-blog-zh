无论你的答案是什么，你都会想知道如何在谷歌文档中快速地做到这一点。你当然可以高亮显示文本，点击右键，然后再点击，最后粘贴你的链接，但你也可以节省一些时间和精力。所有这些微操作确实会增加你的用户体验。

你要做的是突出显示文本，然后在键盘上简单地按下:

> command + k (Mac)
> 
> control + k (Windows)

就是这样！这就是答案。如果你立即去尝试，这是最受欢迎的文章之一。说真的:马上打开谷歌文档。打字什么的。突出它。然后按下`command + k`或`control + k`，然后插入您的链接。再做一次。记住它。收获回报。说真的，有时候就是生活中的小事。
