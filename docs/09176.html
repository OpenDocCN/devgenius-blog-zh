问题:→

给定一棵二叉树的root，返回其节点值的 以便遍历 。

例 1: 

Input: root = [1,null,2,3]
Output: [1,3,2]

例 2: 

Input: root = []
Output: []

例 3: 

Input: root = [1]
Output: [1]

约束:

  • 树中的节点数量在范围[0, 100]内。
  • -100 <= Node.val <= 100

跟进:递归求解很简单，你能迭代吗？

解决方案:→

不像线性数据结构(数组、链表、队列、堆栈等)只有一种逻辑方式来遍历它们，树可以用不同的方式来遍历。

遍历是:
(a)顺序(左，根，右):4 2 5 1 3
(b)前顺序(根，左，右):1 2 4 5 3
(c)后顺序(左，右，根):4 5 2 3 1

在问题中，清楚地提到我们必须以的方式遍历。

让我们先了解什么是以便。

  1. 遍历左边的子树，
  2. 访根。
  3. 遍历右边的子树，
