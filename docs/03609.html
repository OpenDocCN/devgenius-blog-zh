邮递员的环境变量

在我的大多数项目中，我使用 Postman 来检查我的 API 是否工作正常。大多数都使用 Firebase 来验证用户，所以我需要检索 Firebase 提供的会话令牌，并将其附加到每个响应的头中。

如果您手动这样做，对于几个端点来说是可以的，但是如果您有很多端点将这个令牌附加到头部，您将需要很多时间。
