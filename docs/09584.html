1。单引号-双引号困境:

在这里，你会注意到第一个打印的是 AA ，第二个打印的是 130 而不是 AA。

原因是，如果使用双引号，字符将被视为字符串，但在单引号的情况下，字符值操作数(“A”和“A”)将通过一个称为扩大原语转换的过程转换为 int 值。

如果你看到 ASCII 表，

char A 将转换为 int 65，

A' + 'A' => 65 + 65 => 130。

因此，在转换之后，数字返回 130。
