你有没有想过成为一名软件开发人员是不可能的？

当然，这听起来很好，很别致。每个人都认为这些开发人员是无敌的天才，他们知道电脑周围的一切。

事实是，他们没有。

要成为一名优秀的开发人员，你不需要知道软件如何运行的一切。你只需要知道足够多。

什么是足够的？

![](img/4dbf821db3931f82e2efa1d94e573d3d.png)

照片由 [Edho Pratama](https://unsplash.com/@edhoradic?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

如果你对你正在使用的语言有很好的基础知识，并且知道如何用谷歌搜索解决问题，你会做得很好。当然，如果你想成为一名更高级的开发人员，你需要对软件工作的所有部分有很好的理解。

现在，只关注最基本的。

在弄清楚如何成为一名开发人员之前，最好弄清楚你的个人灵感是什么。你是想成为一名后端开发人员来维护服务器 API，还是仅仅编写每个人都能立即看到的 UI 片段？很可能你没有这个问题的答案。

完全没问题。

如果我是你，这将完全归结为流行的语言使用。像 python、java 或 javascript 这样的语言将是一个良好的开端。特别是 Javascript，它将会是一门非常值得学习的语言，因为它在今天有着巨大的需求。react 和 node 等语言都是在 javascript 中运行的流行库。

一旦你想好了要选择什么语言，我们就要想好下一步该怎么做。

嗯……现在怎么办？

如果我对编程一无所知，我可能不知道该学什么。当然，我可以在谷歌上搜索基础 java 或 javascript 开发应该学些什么，但这对你有什么好处吗？

所以我有个提议。一个你可能不喜欢的，因为它要花钱。

![](img/cafb47163caa5af652beabf90ad8727d.png)

没错！导师！

找一个经历过挣扎的专业人士会给你很大帮助。好消息是这可能会把你弹射到你需要的地方。坏消息是…这需要花钱。

但这是值得的。相信我，我已经走过这条路，我无法表达在找到一位导师帮助我解决这个问题后我有多高兴。

我在哪里可以找到这样的人？我个人推荐 wyzant 或者 codementor.io 找对象。如果您已经购买了 udemy 课程，请尝试私下联系讲师。他们可能会回应，并提供一对一的帮助。

找到一个你可以联系的导师。很多人声称他们知道这种语言，他们特别问你他们能解决什么样的问题。但这不是你需要的。你真的不知道从哪里开始，你需要有人可以一步一步地指导你。

确保你的导师提供了一份你需要学习的东西的清单。如果你的导师对一个主题挖掘得太深，那么这可能不是你想要的。你的导师应该专注于教授基本概念(无论是 javascript 还是 python 或 java)以及如何通过公共库解决编码问题(比如使用字符串或数组反转字符串等)。
