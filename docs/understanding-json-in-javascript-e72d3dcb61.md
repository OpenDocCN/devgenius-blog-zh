## 介绍

我们可以说，JSON 是一种非常简单和可读的格式，我们主要使用它来以一种有组织的方式在服务器和我们的 web 应用程序之间传输和存储数据，它使我们更容易访问数据，并且它给我们一组人们可以以一种简单的方式阅读的数据。

## JSON 的元素
