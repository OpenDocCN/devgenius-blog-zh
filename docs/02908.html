Github Gist 支持以下格式:

  • CSV(逗号分隔值)
  • TSV(制表符分隔值)

逗号分隔值

对于 CSV 格式，数据集格式规则。

  • 逗号分隔这些值。
  • 每行必须有相同的列数。
  • 文件扩展名必须是"。CSV "
