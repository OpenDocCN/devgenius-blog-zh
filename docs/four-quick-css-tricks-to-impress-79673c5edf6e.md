# #3 方框阴影/字体阴影

想给你的字体或其他元素增加一些深度吗？使用框阴影或字体阴影将帮助您实现这一效果，这将添加一个可怕的附加，并真正使您的元素弹出。尝试不同的像素大小，颜色和位置，以达到你想要的效果，这可能是微妙的或巨大的！

![](img/c8f28b1dccca5257bc3c4b08c5057e00.png)

框阴影非常适合添加一些 3D 效果。
