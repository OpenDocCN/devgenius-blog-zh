在任何领域成为专家总是需要时间的。

尼廷，你说的专家是什么意思？我来帮你举个例子。

假设我已经创建了一个像 [nitinsharma](https://www.nitinsharma.ml/) 这样的基本网站。看起来更好，但如果你访问这里的。的确，第二个网站足够伟大。

同样的概念也适用于此。一个伟大的网站总是由专家制作的。

现在你怎么能成为那个人呢？简单地通过进一步阅读。

# 1.开放源码

开源是一个很好的资源，你必须把它记在脑子里。WordPress.org、Ubuntu、Debian、Kali Linux 等都是开源的例子。

甚至谷歌也鼓励开发者通过各种项目参与开源，比如 GSoC 等。

当我刚接触 Web 开发时，了解 GitHub 和 Git 有点困难。但也足够简单，也可以从 YouTube 视频、Medium 博客等学习。

# 2.向他人学习网站

经过一个月的学习网页开发，我决定为自己创建一个网站。为此我使用了 HTML、CSS 和 JavaScript。

首先，我在 Figma 设计了我的网站，然后开始创建它。

将 CSS 应用于 HTML 后，我不知所措。Flexbox 不能正常工作，宽度超过 100%，没有任何反应，设计是垃圾。

然后我去看了很多开发者组合网站，对我帮助很大。

我学到了很多颜色的混合，干净和现代的设计，等等。

我一生最喜欢的网站是[布鲁诺-西蒙](https://bruno-simon.com/)和[耐克-反应](https://www.nike-react.com/)。
