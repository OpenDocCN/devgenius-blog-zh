速记员太棒了！在那里，我说了。速记操作符将右边的表达式与赋值操作符组合在一起。出现在赋值运算符左侧的变量也需要出现在右侧，以便您能够使用速记符号。

让我们看看下面的代码:

<?php$x = 1;
$x = $x + 1;echo $x;?>
  1. PHP 将整数值 1 赋给变量$x。
  2. 在第二个语句中，$x + 1 被求值。$x 包含值 1，因此结果将是 1 + 1，等于 2。
  3. 值 2 被分配给$x。

如果您回显$x，将显示 2。

由于变量$x 出现在赋值运算符的两边，我们可以使用速记运算符来缩短表达式$x = $x + 1。您只需从右侧移除$x，并将+运算符移到=运算符前面:$x += 1。

<?php// Long approach
$x = $x + 1;// Can be shortened to
$x += 1;?>

速记操作不限于整数；您可以使用串联简写运算符来组合字符串。

<?php// Before
$msg = "Hey";
$msg = $msg . " there";// After
$msg  = "Hey";
$msg .= " there";?>

我们可以将同样的逻辑应用于减法、乘法，甚至是模运算。

<?php$x = 0;// Same as $x = $x + 4;
// $x = 0 + 4; 
// $x = 4
$x += 4;// Same as $x = $x - 2;
// $x = 4 - 2;
// $x = 2;
$x -= 2;// Same as $x = $x * 2;
// $x = 2 * 2;
// $x = 4;
$x *= 2;// Same as $x = $x % 2;
// $x = 4 % 2;
// $x = 0;
$x %= 2;?>

速记操作符在整个程序设计中经常使用；它们如此频繁，以至于你很少能在野外看到它们。有些操作被如此频繁地使用，以至于产生了更短的操作符。我说的是递增和递减运算符，我们最近已经讨论过了。
