如何在谷歌文档中添加删除线

删除线文本比你想象的要方便得多。

我发现这对于透明性来说很好；有时候，你已经在谷歌文档中写下了一些东西，如果它与其他人分享，最清楚的方法是划掉它，而不是删除它，假装那段文字从未存在过。

换句话说，这是一种表示反对的方式，或者简单地说，以前写的东西现在不适用了。

老实说，有一长串的原因让你想要删除文本，所有这些我都无法在这篇文章中详细描述。n̶o̶̶o̶n̶e̶̶w̶o̶u̶l̶d̶̶r̶e̶a̶d̶̶i̶t̶̶a̶n̶y̶w̶a̶y̶s̶.̶

(顺便说一句，为了某种幽默效果而划掉文本是另一个原因)。

在谷歌文档中，我发现删除线选项并不总是很容易找到。它不在大多数平台的顶部工具栏中(我没有检查所有的平台，但我确实发现它经常不在那里)，它也不是很多人都知道的像command + C或control + C这样复制文本的快捷方式。

那么我们如何在谷歌文档中删除文本呢？

这取决于你用的是MacOS还是Windows。

苹果个人计算机

如果您使用的是MacOS(大多数情况下也就是使用苹果电脑)，请突出显示您想要删除的文本，然后点击以下键盘快捷键:

command + shift + X

Windows操作系统

使用Windows？然后，您需要突出显示想要删除线的文本，然后点击以下键盘快捷键:

alt + shift + 5

Mac或Windows，没有键盘快捷键

如果你不喜欢使用键盘快捷键，那么你可以手动点击删除文本。

为此，进入顶部工具栏的格式。

然后，将鼠标悬停在文本上。最后，你可以点击删除线。您还会看到与您所在的操作系统相对应的键盘快捷键。
