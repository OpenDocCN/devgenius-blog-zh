让我们再来看看上一篇文章中提到的汽车工厂。

想象一个汽车工厂:

  1. 你把材料带到工厂(你在这里)。
  2. 你用这些材料给植物施肥(和这里的)。
  3. 工厂获取原材料，然后以某种方式生产出一辆汽车。

汽车工厂是函数，它可以被调用来生产某种东西。之前创建的 plant() 函数有一个本地变量 $type_of_plant ，它存储了由 plant() 函数回显的植物类型。

如果我们可以使用 plant() 函数，传递给它一些字符串，并让它回显我们传递给它的任何内容，这不是很好吗？我们想给我们的工厂送一张便条。工厂内的员工收到该通知，并能够阅读该通知。这张纸条告诉植物它是什么类型的植物。

这就是争论发生的地方。参数是我们调用函数时传递给函数的数据。让我们更仔细地看一下这个函数的结构。

function function_name( $parameter_1, $parameter_2, ... ) {
  // function body
}

我们在上一篇文章中看到了整体的函数声明。唯一的新概念应该是圆括号中声明的参数。你可能已经注意到，早些时候我在谈论参数，现在我在谈论参数。当我们调用函数时，我们传递参数；当我们声明一个函数时，我们声明参数。相当多的程序员对此感到困惑。

函数可以有多个参数。每个参数由逗号分隔。参数命名约定与变量命名约定相同。

当向函数传递参数时，只需想象您正在为参数赋值。

function_name("Subscribe", "Please");

让我们修改一下 plant() 函数，这样它就可以接受一个参数并且回显它。我们将声明一个名为 plant() 的新函数。该函数将有一个参数$植物类型。该函数将回显分配给 $type_of_plant 参数的任何内容。将会调用两次植物(＄植物类型)函数。每次都会提供一个新的参数。

让我们更详细地看看上面的代码。

  1. PHP跳过了函数声明。
  2. PHP在第7行遇到了 plant("Automotive") 函数调用。
  3. PHP找到被调用的 plant() 函数，并将值“Automotive”赋给第3行的 $type_of_plant 参数。
  4. PHP进入函数体，回应“我是一个汽车工厂。”
  5. PHP到达函数的末尾并返回到第7行。该语句以分号结束。
  6. PHP移动到下一行，遇到下一个函数调用， plant(“蔬菜”)。
  7. PHP找到被调用的 plant() 函数，并将值“蔬菜”赋给第3行的 $type_of_plant 参数。
  8. PHP进入函数体呼应“我是植物人。”
  9. PHP到达函数体的末尾并返回到第8行。该语句以分号结束。

如果没有参数传递给函数 plant() 会怎样？因为它需要一个参数，所以会抛出一个错误。在函数声明期间，可以将默认值分配给 $type_of_plant 参数。如果在没有参数的情况下调用 plant() ，将显示默认值。如果 plant() 被一个参数调用，该参数将覆盖默认值，并且该参数将被显示。为参数分配默认值在语法上等同于为变量分配值($ type _ of _ plant = " Automotive "))。

在上面的例子中，当 plant_two() 被无参数调用时,回显出“我是一个(n)汽车计划”。当 plant_two("Subaru") 被一个参数调用时，显示“我是一个(n) Subaru plant”。

一个函数可以用一些包含默认值的参数和一些不包含默认值的参数来声明。默认值参数必须被推到声明的末尾。您需要的参数必须出现在开头。当使用多个参数时，必须按照声明参数的顺序提供参数。如果默认参数在开头，而你跳过了它们，PHP怎么知道你跳过了什么？它只是看到值进来。有一些编程语言可以在调用函数时显式地将参数赋值给形参，但PHP不是其中之一。事实上，大多数编程语言都遵循与PHP相同的规则。

function func_name($p_1, $p_2, $p_3 = "Default") { }

让我们看一个混合两种类型参数的例子。我们将创建一个名为 car() 的函数。该函数将能够接受三个参数:车辆品牌、车辆型号和车辆传输。将需要$品牌和$型号参数。 $transmission 参数是可选的，默认为“手动”功能体将回应出“您刚刚在变速器中购买了一个品牌型号。”

在第一个 car("Subaru "，" WRX STi") 函数调用期间， $transmission 参数被省略。$传输参数被赋予默认值“手动”顺便说一下，STi只配备手动变速器。斯巴鲁好样的！一串“你刚买了一辆斯巴鲁WRX STi在一个(n)手动变速箱”是呼应出来的。

吉普牧马人有两种选择，购买它的人恰好想要一个自动变速器。调用汽车(" Jeep "、"牧马人"、"自动")函数，并指定所有三个参数。PHP输出“你刚买了一辆自动变速箱的吉普牧马人。”

函数参数是编程不可或缺的一部分。确保你理解他们。
