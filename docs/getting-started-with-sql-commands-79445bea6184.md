# 简介:

SQL 代表**结构化查询语言**。SQL 用于与 RDBMS(关系数据库管理系统)通信(存储数据/管理数据)。它用于在 RDBMS 上执行数据操作。

SQL 遵循不同的方法来操作数据。SQL 中的基本命令分为 3 个命令:

*   数据定义语言
*   DoctorofModernLanguages 现代语言博士
*   TCL 集团股份有限公司（TCL Corporation 的缩写）
