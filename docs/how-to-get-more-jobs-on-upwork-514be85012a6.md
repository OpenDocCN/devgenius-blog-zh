# 如何在 Upwork 上获得更多的工作

> 原文：<https://blog.devgenius.io/how-to-get-more-jobs-on-upwork-514be85012a6?source=collection_archive---------5----------------------->

我是 Upwork 的一名相当新的自由职业者，我学到了一些东西，与其他人相比，这些东西帮助我获得了新客户。对我来说，这段旅程始于两个月前，开始时，我连一个客户都找不到。在应用了我将在下面谈到的技巧后，这种情况发生了变化。现在我有大约 9 到 10 个客户，我正在为他们工作。我甚至不得不拒绝一些新客户，因为我手头的工作太多了。是的，没错，我从一开始几乎一个月没有客户，到过去的一个月有太多客户需要处理。如果你也是这个自由职业平台的新手，并且希望获得更多的客户，那么我希望这篇文章能帮助你。

# 选择你的技能

我想尝试自由写作，因为我已经写了几年博客了，而且我越来越擅长了。此外，在写文章之前，我会做一些令人惊讶的研究工作，这也是我坚持下去的动力。我主要在我的博客上工作，我没有在其他主题上工作过，所以一开始有点挑战性，但我很快就找到了适合我的技能。我从我以前做过的事情中选择，我知道如果我有正确的动机，我可以发展得更多。到目前为止，这些技能对我来说非常有用。

# 把你的工作当成你的事

你在 Upwork 上的个人资料是你的潜在买家会看到的，所以你应该尽可能多地填写你之前做过的其他相关项目或工作的信息。您也可以为您选择的特定技能创建专门的个人资料，例如，我仍在我的“开发人员”个人资料中列出我以前做过的[项目和计划](https://levelup.gitconnected.com/what-i-learned-from-developing-my-first-application-54cc608e58d8)。这是客户看到你如何准备和专业，这是他们与你做生意的切入点。如果他们喜欢第一印象，十有八九会给你写信。但让他们和你做生意的不是这个。

# 专业点

从你通过求职信与他们接触的那一刻起，你就开始变得专业。在那里你必须展示你的写作实力和能力。保持谦逊，总是以正式的方式交谈。如果你想长期留住客户，那就坚持你的时间表，总是按时交货。最重要的是要精确，要有高质量的工作。遵循这些想法对我很有效，我每天都在继续成长和学习。一份工作没有完美的封面或候选人，你只需要不断努力，找到最适合你的。

# 连接系统

就与客户和自由职业者打交道而言，这是自由职业平台中较好的系统之一。在这一点上，这是一个开放的市场，用户没有任何真正的方法可以“欺骗”这个系统。你必须小心你宝贵的人脉，确保选择你热爱的项目，这样工作才不会让你感到无聊。我设法在一开始就找到了一个客户，而不必从他们那里购买更多的连接，从那个项目的收益中，我购买了会员资格，这转向了更多的连接，最终更多的客户。所以我没有仅仅凭借我在平台上赚的钱用个人卡购买它们。

# 获得更多客户的技巧

确保你了解你愿意提交申请的工作。如果你觉得有些不对劲，那就对客户进行调查，看看他以前是否提供过更多的工作机会。

让客户明白为什么你很适合他们提供的工作，如果他们问，总是展示一些你以前的工作。

确保你有关于这份工作的所有必要信息，这样你就知道什么该做，什么不该做，这样你的工作就不会被拒绝。

几乎每时每刻都要与客户沟通，这样你才能理解他们的期望，以及是否有其他事情发生。如果你不能按时完成工作，最好通知客户，以便他们做出相应的计划。
