低级与高级编程语言

原文：https://blog.devgenius.io/low-level-vs-hight-level-languages-1d541652068e?source=collection_archive---------4-----------------------#2021-11-13

T he术语“低级”和“高级”本来就是相对的。低级编程语言是一种从计算机指令集体系结构中提供很少或不提供抽象的编程语言。单词“低”指的是语言和机器语言之间的少量或不存在的抽象；正因为如此，低级语言可以被转换成机器代码，而不需要使用编译器或解释器，结果代码直接在处理器上运行。

高级编程语言是一种对计算机细节进行高度抽象的编程语言。与低级语言相比，它可能使用自然语言元素，更容易使用，或者跨平台更易移植。这种语言隐藏了CPU操作的细节，如内存访问模型和范围管理。

一种编程语言，如C语言或Pascal语言，使程序员能够编写或多或少独立于特定类型计算机的程序。这样的语言被认为是高级语言，因为它们更接近人类语言，例如:' Python '而更远离机器语言

与低级语言相比，高级语言的主要优势在于它们更容易阅读、编写和维护。用高级语言编写的程序必须由编译器或解释器翻译成机器语言。
