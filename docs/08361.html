如果在 Python 中 else

如果用 Python 中的 else 来做决策，那么当我们想只在满足某个条件的情况下执行一个代码时，就需要用到它。

value = 'apple'
if value == 'apple':
    print('True')
else:
    print('False')
Output- 
True

elif 是 else if 的缩写。它允许我们检查多个表达式。

value = 'apple'
if value == 'apple':
    print('True')
elif value == 'Grapes':
    print('Green')
else:
    print('False')
Output- 
True

逻辑运算符

Python 有三个逻辑运算符。

and 运算符检查这两个条件是否同时为True。如果两个条件都是True，则返回True。如果任一条件为False，则返回False。

与and操作器类似，or操作器检查多个条件。但是当一个或两个单独的条件都是True.时，它返回True

While 循环

Python while 循环允许执行部分代码，直到给定条件返回 false。

a = 5
while a < 10:
    print(a)
    a += 1

中断并继续语句

当遇到 break 语句时，它将控制带出循环，当遇到 continue 语句时，控制转移到循环的开始。

a = 1
while a < 10:
    a += 1
    if(a == 6):
        continue
    elif(a == 8):
        break
    print(a)

2
