我会为你提供一些真正实用的解决方案，解决你不会写代码的问题。

首先，编码就像一个游戏，任何人都可以学习，如果他们有兴趣创造一些有成效的东西。

作为一名优秀的程序员，你实际上可以做出很多贡献，让事情变得更简单、更好。

我认为这可能有两个原因:

## 1-你没有基本的理解

我将无法建立一个网站，除非我有一个像 HTML，JavaScript，jQuery 等基本 UI 技术的良好知识。

## 2-你理解编码，但缺乏实践

编码就是练习，你练习的越多，你就越擅长。

所以作为一个解决方案，我建议从基础开始，给自己一些时间去吸收新的概念，并不断练习每个概念，直到你有足够的信心。

作为一名 Java 和 Javascript 开发人员，我建议您遵循:

javadocs，第一本 Java 书籍《干净的代码:Java 编程语言的敏捷软件工艺手册》的负责人。和 Mozilla docs for Javascript。

例如，如果你试图同时从多个来源学习 Java，那么你将会花费更多的时间来学习一个概念。

在尽可能多的基础知识上花足够的时间，因为一旦你开始学习与 Java 相关的高级技术，这是唯一能提高你的编码技能的东西。

下面是一些技术和语言的列表，您可以根据自己的学习需求进行调整:

*   如果你想开发一个简单的静态网站:按顺序学习 HTML5，CSS，JavaScript。
*   想要建立一个动态响应的网站:例如学习 Angular 和 Bootstrap。
*   想搭建一个简单的应用:学习核心 Java，。net 还是 Python
*   想要构建企业应用:学习高级 java ( JSP、Servlets、Struts、Spring Framework、Web 服务)

# 结论:

从大处着眼，从小处着手。

开始每天练习代码，并尽可能快地做得更好。
