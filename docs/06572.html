1.macOS上的Python设置

要在macOS上安装Python，请遵循以下步骤:

  1. 进入Python官方网站Python.org，从下载下拉菜单中选择 MacOS X 。参见图1。

2.选择Python版本，下载 macOS 64位/32位安装程序 ( pkg文件)。参考图2。

3.一旦 pkg文件下载到你的电脑上，打开下载的文件。现在你可以看到macOS安装向导在你的电脑上安装Python。

4.您必须同意许可协议并继续点击继续，直到安装程序要求您输入您的用户密码，输入您的密码，然后点击安装软件。参考图3。

5.安装程序在您的电脑上成功安装Python后，点击关闭。

6.您将看到一个对话框，询问您是否要保留或删除安装文件，选择您想要的。参考图4。

现在我们将测试Python是否安装成功

  • 打开你的应用文件夹，检查是否有 Python 。参考图5。
  • 第二次打开您的终端窗口并键入:
python3 --version

点击回车。

您应该获得Python版本。参见图6。

让我们继续安装PyCharm IDE。

2.MacOs上的PyCharm设置

简单来说，按照步骤:

  1. 从您的互联网浏览器访问 PyCharm下载页面 。
  2. 在Mac下，从下载页面选择社区版本。然后点击下载。参考图7。

3.一旦 dmg 文件下载到你的电脑上，打开下载的文件。

4.在安装程序完成打开并验证您的PyCharm dmg 文件后，您应该将 Pycharm文件拖放到您的计算机应用程序文件夹中。 参考图8。

5.当安装程序将 PyCharm 复制到你的电脑应用程序文件夹后，点击文件夹中的PyCharm图标，然后点击打开。参考图9。

6.接受JetBrains隐私政策并点击继续。参考图10。

7.共享或不共享您环境中的一些数据由您决定。参见图11。

8.现在我们可以开始定制我们的PyCharm了。选择 您喜欢的键映射方案，然后点击下一步。参考图12。

9.选择您喜欢的 UI主题，然后点击下一步。参见图13。

10.选中复选框(创建脚本…)，然后点击开始使用PyCharm。参考图14。

11.安装程序要求您输入用户密码，输入密码，然后点击确定。参见图15。

出现一个欢迎窗口，您可以开始并创建您的第一个项目。参见图16。

恭喜！
