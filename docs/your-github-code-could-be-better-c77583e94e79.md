# 以及该怎么做

学习软件工程实践通常有两种形式:

*   教育内容——书籍、课程、讲座等。
*   代码审查

然而，**差距**怎么办？也许一本书漏掉了一个常见的编码错误，或者一个代码审查员漏掉了一个重复出现的错误。**找到代码漏洞最简单的方法是什么？**

静态代码分析器。是的，我们将利用技术使我们的技术变得更好。这很容易。

***介绍 LGTM。*** [塞姆勒的 LGTM](https://lgtm.com/) 于 2019 年被 GitHub 收购。你需要做的就是把你的公共 GitHub 库链接放进去。您将获得代码质量分数和建议。

LGTM 会做持续的分析，寻找漏洞和代码改进，所以你不必。您可以为您的存储库设置监控，甚至拉取请求。结果，你会得到*对你的代码的建议，你可以从*中修正和学习。更高级的用例包括编写自己的查询。

给 LGTM 一个机会，评论你的发现！
