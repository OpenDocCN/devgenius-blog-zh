# **1。选择多生态位而不是单一生态位。**

我知道一个技术人员可以写关于人工智能、UI/UX 设计、AR/VR 等技术的文章，但他/她也可以写关于商业、创业、领导力、营销、自由职业者、远程工作、创造力、金钱、最好的编码书籍，这是无限的。

> 我的意思是选择相互关联的多领域。
