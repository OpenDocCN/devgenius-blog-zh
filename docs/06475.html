1.Linux上的Python设置

要在您的Linux上安装Python，请遵循以下步骤:

  1. 用 Ctrl+Alt+T 打开终端并键入:
sudo apt-get install python3.6

点击进入并确认输入3。不然就要装Python 2.7了。

2.之后，你必须输入当前用户的密码。

3.完成后，只需输入:

Python3.6

然后进入。

4.如果你找到了 Python 3.6 ，那意味着你现在已经准备好使用Python了。参见图1。

5.最后，通过运行 Ctrl-z 退出Python shell，然后进入。

让我们继续安装PyCharm IDE。

2.Linux上的PyCharm设置

简单来说，按照步骤:

  1. 从您的互联网浏览器访问PyCharm官方网站 PyCharm下载页面 。
  2. 在Linux下，从下载页面选择社区版本。然后点击下载。参考图2。

3.一旦tar.gz文件下载到你的电脑上，解压下载的文件。参考图3。

4.一旦它被提取，你会发现多个文件夹和文件。
有一个文件，在 bin文件夹里面，它的名字 PyCharm.sh 。参考图4。

5.启动位于bin文件夹中的 pycharm.sh 文件脚本。为此，请打开您的终端并导航到脚本文件位置。

cd Downloads/pycharm-community-2020.1.2./pycharm.sh

然后按回车。参考图5。

6.接受隐私政策并点击继续。参考图6。

7.共享或不共享您环境中的一些数据由您决定。参见图7。

8.现在我们可以开始定制我们的PyCharm环境了。选择你喜欢的 UI主题，然后点击下一个。参见图8。

9.选中复选框(创建脚本…)，点击下一步。参考图9。

10.你可以下载并安装任何PyCharm插件或者跳过它，点击开始使用PyCharm 。参见图10。

11.安装程序会要求您输入用户密码。输入密码，然后点击确定。将出现一个欢迎窗口。在这里，您可以开始并创建您的第一个项目。参见图11。
