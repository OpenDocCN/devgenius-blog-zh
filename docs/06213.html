Python 概述

ython 是一种编码语言，由吉多·范·罗苏姆在 20 世纪 90 年代早期发明。它是五种最流行的编码语言之一。当人们谈论编码时，它是最先被提及的编码语言之一。

也许你听说过，或者你已经知道一点了。这里我将提到一些你绝对应该知道的关于令人兴奋的编码语言 Python 的信息！

有趣的事实:发明者并不是因为蛇才称之为 Python。他之所以称之为 Python，是因为英国喜剧演员组合蒙蒂 Python 。

Python 的优势

P ython 是多功能编码语言。它被用于计算机科学，用于创建图形用户界面，用于网络开发— 仅举几个使用它的领域。它还专注于编写可读代码(我们将在本系列的后面回到这一点)，这就是为什么共享代码通常非常容易。除此之外，它还有一个非常好的可移植性，这意味着你可以在许多平台上运行它。比如 Windows，Linux，Macintosh，Android，iOS 等等。还有许多库，也称为模块(我们将在本系列的后面回到这一点)，它们允许你使用已经编写好的代码片段，所以你可以很容易地在你的代码中实现复杂的东西。另一个奇妙的事情是 python 是快速可写的。这意味着您不必编写一千行代码，只需打印一些东西即可——有一个名为“Hello World”的示例，我将在本系列的稍后部分向您展示，它将向您展示我的意思。还有一个非常大，活跃的 和成长中的社区会在任何情况下帮助你。

但是最伟大的一点就是对于初学者来说很好学！所以你学 Python 不会像你想象的那么难。

一般来说什么是编码？

但是让我们回到起点！我听到一些人说:“但是什么是编码？”—让我尽可能简单地给你解释一下。

你输入一个东西，叫做 (源)代码， 运行它，计算机翻译它 ， 你得到一个输出或者最终结果。让我们以烹饪为例:

  • 把牛奶放进碗里
  • 给它鸡蛋
  • 给它一些胡椒和盐
  • 混合在一起
  • 在锅里煮它

炒鸡蛋准备好了！

你也给计算机一些指令，但是像这样的指令不够清楚，我们需要一种方法来清楚地与计算机交流。我们通过向计算机“发送”0 和 1 来做到这一点，计算机将读取这些数据并做我们希望计算机做的事情。为了让人们更容易理解，有很多编码语言，比如 C++、Java、JavaScript 等等，还有 PYTHON！这些编码语言将人类编写的代码翻译成带有 0 和 1 的机器语言。

这就是简而言之的编程背后的全部“魔力”!所以程序员并不是真正的魔术师，他们每分钟打 1000 个单词，可以在 10 秒钟内黑掉你！它更像是绘画或诸如此类的东西。你创造了美好的东西！这就是为什么我和许多其他人喜欢编码。是的，当然，当你会编程并能挣很多钱的时候，你可以得到一些真正好的工作。
