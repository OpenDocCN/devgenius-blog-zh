# 软件开发人员应该关心的非常简单的方面

## 成为一名优秀的团队成员

![](img/7304e52f1d5c5dddaffe214004a5f096.png)

安东尼·里埃拉在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

最近，我看到很多人可以写 10 行代码，并开始称自己为专业软件开发人员。

首先让我澄清一个事实，你不需要一份工作就可以称自己为软件开发人员。所以不要仅仅因为你在 IT 公司有一份工作，就一直在城市里闲逛，告诉别人你是一个真心实意的软件开发员。

以下是我认为对于一个真正的软件开发人员来说最重要的几个方面。另外，你不需要花太多力气就能全部学会。

## 管理的艺术

我见过许多人，他们可以写出非常好的代码，但是当他们真正需要共享他们的代码时，却找不到代码的路径。给自己创建一个名为**我的代码**的文件夹…

此外，我见过有人写了一个漂亮的代码，当客户想要修改程序时，这个人就完全卡住了。伙计，你是不是刚刚从 GitHub 复制了代码…永远对你的项目和客户保持诚实。记住一个简单的句子“**现在抄，以后吃**”

请保持你的代码的真实性，你应该知道你写的代码的每一部分。

## 沟通

成为一名软件开发人员并不意味着你需要在笔记本电脑上输入代码，除非你拥有一家大公司。许多开发人员在两个人的团队中工作，不能自由地互相寻求帮助…

首先，软件开发人员为他们的客户或公司写代码，你需要严肃的沟通技巧。安静地坐着不会带你去任何地方。

如果你像我一样是一个内向的人，那么试着给你的队友发短信，进一步试着和他们建立稳固的联系。

## 还是学生

我见过一些人只是 HTML 或 XYZ 语言的专家。好吧，我们同意你是某一种语言的专家或专业程序员，但是这个世界的变化比你想象的要快。

如果你知道 HTML，试着学习 CSS 和 JavaScript…

如果您是全栈开发人员，请获得 AWS 认证…

只要不断提升人们，我们就处在一个只有能够学习新技能的人才能生存的领域。

我希望你喜欢这篇文章，只是要记住成为一名优秀的软件开发人员非常重要。相信自己，不断升级。祝你好运！
