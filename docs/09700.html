另存为 index.html。为什么索引？网络服务器知道寻找索引文件作为主文件。如果你把你的网站上传到主服务器文件夹，通常是 public_html ，服务器软件会寻找 index.html 或 index.php的文件。因为这是默认文件名，所以当您输入地址时，不必在地址栏中指定文件名。

比如可以输入 dinocajic.xyz 或者dinocajic.xyz/index.html；他们会把你带到同一个页面。文件index.html应该是小写的，因为一些服务器不知道寻找大写版本。你的其他文件也应该是小写的。如果文件名由多个单词组成，请使用下划线或破折号作为分隔符。例如，如果您正在创建“关于我们”页面，您应该将该文件命名为 about_us.html 或 about-us.html。
