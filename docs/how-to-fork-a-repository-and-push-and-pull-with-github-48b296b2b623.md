**什么是 GitHub？**

**GitHub** 是一个 Git 存储库版本控制系统，您可以在终端工作时使用。**版本控制**允许任何人在项目上同时协作，对文件进行修改，并保存或**“提交”**。Git 跟踪对文件所做的更改，这样您就有了所做工作的记录，并且如果需要的话，您可以恢复到文件的以前版本。

> ***先决条件:*** GitHub 账号
> CentOS7 服务器
> 命令行终端
> 一个拥有 Sudo 访问权限的用户账号
> Vim

在本教程中，你将学习如何…
1。安装并配置 Git
2。在 GitHub
3 中派生一个库。使用 GitHub 推送文件
