# 数据类型

在 Swift 中，变量为我们提供了一个我们的程序可以处理的名字。每个变量都有特定的类型，可以决定变量的内存和大小。

Swift 中的一些基本变量类型:

*   字符串—一个`collection of character`。
*   Int —用于`whole number.`
*   Float —代表一个`32-bit`位浮点数。
*   Double —表示一个`64-bit`浮点数。
*   布尔—表示为`true`或`false`
*   字符— `single-character`字符串。

Swift 还允许定义其他类型，如**数组**、**字典**、**可选**、**结构**和**类**。
