概观

函数接口是只包含一个抽象方法的接口。从 Java 8 开始，lambda 表达式可以用来表示函数接口的实例。一个函数接口可以有任意数量的默认方法。

在这篇文章中，我想推荐 Java 8 中最常用的四个函数接口，以及独立应用它们和 lambda 表达式的方法。
