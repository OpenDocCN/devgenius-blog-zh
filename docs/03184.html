编程是一个需要更新的领域，每天都需要学习新的东西。

所以，问题是要有大量的信息资源来提高你的技能。

我发现很多人询问YouTube频道，这可能对他们有所帮助。

在这篇文章中，我将与你分享4个YouTube频道，它们在我的编程之旅中帮助了我很多。

1-用Mosh编程

这是学习如何设计和编程的一个很好的youtube频道。

Mosh Hamadani是一名软件工程师和web开发专家。

我喜欢他教授如何构建、管理和理解语言、框架、模式等的方式

2-Udacity ( Udacity )

在线教育的先驱Udacity正在建设“硅谷大学”，这是一种新型的在线大学，教授当今行业雇主需要的实际编程技能。

3-level up tuts(level up tuts

这是PHP和WordPress教程的一个很好的资源。

4-freeCodeCamp.org(自由代码营)

在自由代码营的YouTube频道上有大量2分钟的白板讲解，解释各种软件工程工具和概念。

结论:

观看视频可以欺骗你，让你觉得你的编程技能正在进步，但这是对实际花时间编程的补充，而不是替代。
