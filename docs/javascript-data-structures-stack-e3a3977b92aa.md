# 堆

![](img/1e3b8324f6174619fef56685c59c163e.png)

堆

堆栈是保存数据的线性数据结构。栈的关键是**后进先出。**这意味着最后一个数据进入堆栈，然后从堆栈中出来。

让我们看看 JavaScript 中的堆栈是什么样子的:

*   **push(data):** 这个函数将新数据推到堆栈的顶部。
*   **pop():** 这个函数从栈顶弹出数据并返回。
*   **top():** 该函数返回排名靠前的数据。
*   **isEmpty():** 检查堆栈是否为空
*   **size():** 返回堆栈的大小
