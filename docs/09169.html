让你为未来做好准备的五种免费网络开发资源

这里是我们学习 web 开发的五大免费资源，包括 HTML、CSS、JavaScript 等等。

2022 年到了，这是疯狂的一年。我们已经看到了太多令人惊叹的 科技进步 ，很难相信 2022 年已经过了中期。那么，我们该何去何从呢？

web 开发人员的新机会每天都在稳步涌现，但是这篇文章将为您提供一些最好的资源，让您了解来年将会发生什么。为了在 2022 年及以后成为一名成功的网页开发者，你必须了解最新和最大的趋势。有许多资源可以教开发者 HTML5、Javascript、C++和其他程序的所有新发展。但是这篇文章为那些渴望学习他们需要知道的东西以便在 2022 年成为一名专业的 web 开发人员的人突出了一些最好的资源。所以让我们开始吧。

№1 —学习网站和课程

  • freeCodeCamp 做的一切都是免费的。他们有你能想到的几乎每一个话题的课程和文章。
  • CSS 窍门 有关于 CSS 和许多其他主题的文章和教程。
  • 西 Bos 有一门超赞的 JavaScript 课程。它叫做 JavaScript 30 ，它有 30 个你可以在 30 天内构建的项目。
  • Flavio Copes 有一个关于各种开发主题的惊人博客。

№2 —书籍

我推荐两本书，这两本书都是免费的。 雄辩的 JavaScript-第三版 和 JavaScript:权威指南 这些其实都是很棒的书。

№3 —播客

  • 我听的播客有几个，但我最喜欢的是 句法 FM 。主持人是韦斯·博斯和斯科特·托林斯基。它们涵盖了广泛的 web 开发主题。
  • 我喜欢的其他播客是 商店脱口秀 和 瓢虫 (这是一个面向开发人员的全女性主持的技术播客)。

№4 — YouTube

我相信你已经听说过许多这样的渠道。他们是，

  • 旅行媒体
  • freeCodeCamp (长达数小时的教程)
  • 开发编辑
  • 设计课程 (了解 UI 和 UX 设计)
  • Web dev 简体
  • LevelUpTuts
  • 凯文·鲍威尔 (擅长任何与 CSS 相关的事情)
  • 弗拉维奥
  • 弗罗林流行
  • 泰勒·波茨
  • 韦本法

№5 —社区

所以，当你有一个问题，你不能解决它，你去哪里？

  • 嗯，当然是 Google，但是当你搜索主题的时候，十有八九会看到 栈溢出 的链接。堆栈溢出有所有的答案。如果您有任何问题，很可能已经在 StackOverflow 上得到回答。
  • 然后是 MDN Web Docs ，Mozilla Web Docs 实际上有关于 HTML、CSS、JavaScript 和许多其他主题的教程和文档，但我主要用它们来编写 JavaScript 文档。

所以我希望这份免费资源清单能在你的网络开发之旅中帮助你。T19】
