面向对象的JavaScript——多态性和原始值

JavaScript部分是面向对象的语言。

要学习JavaScript，我们必须学习JavaScript的面向对象部分。

在本文中，我们将介绍JavaScript的面向对象部分。

我们还看对象的基本构件，即原始值。

多态性

多态性意味着，如果我们用一个子类替换另一个子类，如果代码使用父类的属性或方法，程序仍然应该以同样的方式工作。

如果它们都继承自同一个父类，那么它们应该有相同的方法，所以它们应该以相同的方式工作。

如果Person类有talk方法，我们创建了2个继承它们的子类。

然后我们可以使用那些子类中的任何一种‘T2’方法。

原始数据类型

面向对象JavaScript的基本构件是原始数据类型。

在JavaScript中，原始数据类型是字符串、数字、布尔、null、undefined和bigint。

它们都被记录为值，并且没有自己的方法，除非它们被包装在包装对象中。

变量

变量用于存储数据。

它们是具体价值的占位符。

使用变量比使用值来处理实际数据更方便。

这是因为它们可以多次使用。

值只能使用一次。

使用变量需要2个步骤。

我们必须声明它，然后给它赋值。

要声明一个变量，我们可以写:

let a;

变量可以是字母、数字和下划线字符的组合。

但是我们不能从一个数字开始。

然后为了初始化它，我们给它赋值。

例如，我们可以写:

let a = 1;

我们可以一步到位。

用const声明的变量需要在声明时赋值。

所以我们写了这样的东西:

const a = 1;

我们可以在一个语句中声明多个变量，比如:

let a = 1,
  b = 2,
  c = 3;

我们也可以在变量中使用$。

变量名可以以$开头，也可以在其他位置。

$常用作函数或属性名。

变量区分大小写

JavaScript变量区分大小写。

所以如果它们是不同的案例，那么它们就被认为是不同的。

经营者

运算符可以接受一个或多个值或变量，进行运算，然后返回值。

JavaScript有各种各样的操作符，比如算术操作符、位操作符等等。

将两个数字相加或连接字符串。

-做减法。

*做乘法。

/礼部。

%返回一个数除以另一个数的余数。

++将数字递增1。

--将数字减1。

例如，我们可以写下2个数相加:

const sum = 1 + 2;

还有一些复合运算符。

它们使我们的代码更加紧凑。

他们同时做算术和作业。

+=在现有数字上添加一个数字。

-=从现有数字中减去一个数字。

*=将一个数乘以一个现有数。

/=将一个数除以一个已有的数。

%=得出左数除以右数的余数。

例如，我们将一个数字加到一个变量上，写为:

let x = 1;
x += 2;

我们创建了一个变量x，并将其设置为1。

然后我们用+=给x加2。

所以最后x是3。

结论

多态性是指我们可以用一个子类替换另一个子类，并且仍然可以使用父类的方法。

变量保存值，我们可以对它们进行各种操作。
