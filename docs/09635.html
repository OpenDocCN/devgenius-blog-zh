问题:→

给定大小为n的数组nums，返回多数元素。

多数元素是出现超过⌊n / 2⌋次的元素。您可以假设多数元素总是存在于数组中。

例 1: 

Input: nums = [3,2,3]
Output: 3

例 2: 

Input: nums = [2,2,1,1,1,2,2]
Output: 2

约束:

  • n == nums.length
  • 1 <= n <= 5 * 104
  • -109 <= nums[i] <= 109

追问:你能在线性时间和O(1)空间中解决这个问题吗？
