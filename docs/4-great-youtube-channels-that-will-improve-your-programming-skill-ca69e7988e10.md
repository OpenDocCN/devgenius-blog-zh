编程是一个需要更新的领域，每天都需要学习新的东西。

所以，问题是要有大量的信息资源来提高你的技能。

我发现很多人询问 YouTube 频道，这可能对他们有所帮助。

在这篇文章中，我将与你分享 4 个 YouTube 频道，它们在我的编程之旅中帮助了我很多。

## 1-用 Mosh 编程

这是学习如何设计和编程的一个很好的 youtube 频道。

Mosh Hamadani 是一名软件工程师和 web 开发专家。

我喜欢他教授如何构建、管理和理解语言、框架、模式等的方式

## 2-Udacity ( [Udacity](https://medium.com/u/2929690a28fb?source=post_page-----ca69e7988e10--------------------------------) )

在线教育的先驱 Udacity 正在建设“硅谷大学”，这是一种新型的在线大学，教授当今行业雇主需要的实际编程技能。

## 3-level up tuts([level up tuts](https://medium.com/u/502430226ebb?source=post_page-----ca69e7988e10--------------------------------)

这是 PHP 和 WordPress 教程的一个很好的资源。

## 4-freeCodeCamp.org([自由代码营](https://medium.com/u/8b318225c16a?source=post_page-----ca69e7988e10--------------------------------))

在自由代码营的 YouTube 频道上有大量 2 分钟的白板讲解，解释各种软件工程工具和概念。

# 结论:

观看视频可以欺骗你，让你觉得你的编程技能正在进步，但这是对实际花时间编程的补充，而不是替代。
