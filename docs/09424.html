为什么安全意识培训很重要

原文：https://blog.devgenius.io/why-security-awareness-training-is-important-f515c1d56e56?source=collection_archive---------19-----------------------#2022-08-19

今天，我首先阅读了一篇关于 CloudFlare 和 Twilio 如何被钓鱼和入侵的文章。他们要求所有员工都启用多因素身份认证，但攻击者仍然可以进入。怎么会？

入侵 Twilio 并以 Cloudflare 为目标的网络钓鱼者也很容易得手

至少有两家安全敏感型公司-Twilio 和 cloud flare-成为了高级网络钓鱼攻击的目标…

arstechnica.com

TLDR:攻击者创建了一个网站克隆，并发出了一封钓鱼邮件。他们等着个人上当。当用户点击电子邮件时，他们被定向到攻击者的域，并在那里输入他们的凭据。黑客随时待命，自动输入合法域上的凭证。这引发了 2FA 爆炸。用户在攻击者的网站上输入 MFA 代码，黑客立即在合法服务的网站上输入该代码。这使他们能够绕过 MFA 程序，并违反了一些帐户。

这说明了什么？可见 SAT(安全意识培训)的重要性。坏人会想出办法进来，这取决于链条中最薄弱的一环，即用户，来帮助阻止他们进来。

安全意识培训是整体安全策略的重要组成部分。它可以帮助员工识别和避免网络钓鱼陷阱，教会他们正确使用密码的重要性，并帮助他们学会识别电子邮件或网站上的恶意链接。重要的是，你不要给员工提供过多的信息，而是专注于给他们足够的信息，这样他们就可以对他们的在线活动做出明智的决定。

安全意识培训是您整体安全策略的重要组成部分。

这是抵御网络攻击和其他类型攻击的第一道防线，但它也有助于员工彼此保持一致，并确保他们了解最新的威胁和可用的工具。

安全意识培训应该由知道自己在说什么的人来进行，这些人也知道如何从参与者那里获得结果。如果你想从你的努力中获得最大的影响，考虑聘请一位专家作为你的团队或部门的一对一指导者。

有一些软件，如knowe 4已经在他们的系统中实现了 SAT。你设置用户和频率，然后自动推出。

它可以帮助员工识别和避免网络钓鱼陷阱。

进入你的网络的第一步是让员工打开带有链接的电子邮件或短信。如果他们这样做，很可能会被引入网络钓鱼陷阱。

安全意识培训帮助员工识别这些类型的骗局，并通过识别何时有人通过脸书或推特等社交媒体界面以及通过看似合法但可能是假的电子邮件询问敏感信息来避免这些骗局。

它帮助员工学会识别电子邮件或网站上的恶意链接。

你最不希望的事情就是让你的员工成为旨在窃取个人信息的网络钓鱼邮件的牺牲品。事实上，最近的一项调查发现，73%的员工认为他们在过去的一年中成为了网络钓鱼攻击的受害者，三分之一的人表示他们收到了一封要求他们提供密码或信用卡号码等私人信息的电子邮件。

如果你担心你的员工成为网站上的恶意链接或试图进入你的网络的黑客发送的电子邮件的受害者，那么安全意识培训是关键。

不要给员工提供过多的信息，这一点很重要。

你不应该给你的员工提供过多的信息。不要给他们太多，让他们不知所措，或者不明白你想教他们什么。此外，注意不要给他们过多的信息，这些信息与安全意识培训及其对公司员工的重要性完全无关，即使他们可能还不知道。

重要的是，你不要给员工过多的信息，以至于事后没人记得；如果他们觉得自己学到了一些东西，但在最需要的时候却不能应用，这可能会让他们走上自我破坏的道路。

通过安全意识培训，你可以确保组织中更多的人在网络安全基础知识方面达成共识。

安全意识培训是任何组织网络安全工作的重要组成部分。这有助于确保您团队的所有成员在基本网络安全实践方面保持一致，包括:

  • 了解他们的设备是如何工作的，以及它们是如何被入侵的；
  • 了解他们的环境可能会受到什么类型的攻击，以及他们可能会如何应对；和
  • 知道如何报告可疑活动。

结论

谈到网络安全，安全意识培训对任何组织来说都可能是一个游戏规则的改变者。当您的员工知道要寻找什么以及如何实时发现潜在威胁时，您就更有可能避免被黑客或网络罪犯利用。这就是为什么定期提供培训课程如此重要，以便每个人都能从他们的主管和经理那里获得相同的信息，了解在工作中(和外部)使用技术时应如何遵循最佳实践。
