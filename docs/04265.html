有用的免费 API——更多商业 API

原文：https://blog.devgenius.io/useful-free-apis-more-business-apis-45668a8c88a2?source=collection_archive---------5-----------------------#2021-02-17

在软件开发领域，熟能生巧。所以要尽可能多的找方法练习编程。有了免费的公共 API，我们可以通过创建使用这些 API 的应用来练习编程。

在本文中，我们将研究一些可以使用这些 API 的实践项目想法。

谷歌分析

Google Analytics API 允许我们以编程方式配置和分析网站分析数据。

这个 API 下有很多 API。

核心报告 API 允许我们查询维度和指标来创建定制的报告。

嵌入式 API 允许我们在第三方网站上创建和嵌入仪表板。

多渠道漏斗报告 API 让我们查询流量来源路径，导致用户的目标转换。

实时报告 API 让我们能够报告在我们的资产上发生的活动。

元数据 API 让我们可以访问 API 维度和指标及其属性的列表。

管理 API 让我们可以查看和管理帐户、属性、视图、过滤器、上传、权限等等。

用户删除 API 允许我们删除与用户 ID 相关的 Google Analytics 属性中的所有数据。

这些文件可以在 https://developers.google.com/analytics/的找到。

邮箱验证器

MailboxValidator API 允许我们验证电子邮件地址，以提高投递率。

这些文件可以在https://www.mailboxvalidator.com/api-single-validation获得。

我们可以一次验证一个地址，检查一次性电子邮件地址，并检查电子邮件地址是否来自免费电子邮件提供商。

访问此 API 需要 API 密钥。

客户端库可用于 PHP、Java、VB.NET、C#和 Python。

mailgun API

mailgun API 是一个面向开发者的电子邮件服务。

这些文件可以在 https://documentation.mailgun.com/en/latest/获得。

我们可以用它来发送和接收电子邮件，并跟踪事件。

电子邮件可以用各种协议发送。

也可以排队，批量发送。

也可以用这个 API 安排交货。

我们还可以用它来跟踪邮件的打开、点击、取消订阅、垃圾邮件投诉、失败、交付等等。

有些功能只有在我们为 API 付费时才可用。

访问此 API 需要 API 密钥。

马克拉皮

markerapi 允许我们搜索商标。

我们每个月可以免费搜索 1000 次。

为了获得更多的搜索请求，我们每个月要支付 100 美元。

API 文档在https://markerapi.com/。

访问此 API 需要 API 密钥。

Ticksel

Ticksel API 让我们可以跟踪我们的网站。

文件在 https://www.ticksel.com/的 T2。

免费的 API 让我们一个月跟踪 5000 次页面浏览。

我们可以用免费计划追踪无限的网站。

还提供 12 个月的数据保留期。

可以添加 10 个用户或会话标签进行跟踪。

所有请求都是通过 SSL 发出的。

特雷罗

Trello 是一个任务管理软件，我们可以用它来轻松地跟踪项目中的任务。

项目被记录在卡片板上。

API 文件在 https://developer.atlassian.com/cloud/trello/的。

我们可以用带电源的附件来扩展它。

API 认证是通过 OAuth 完成的。

REST API 允许我们读取和更改动作、公告板、清单、成员、通知和反应数据。

每个实体都有许多可用的字段，包括徽章、标签、URL、订户等等。

结论

我们可以用 Google、Trello 和其他 API 做很多事情。

他们让我们发送电子邮件，跟踪任务，访客，等等。
