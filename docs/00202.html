十大软件工程播客

原文：https://blog.devgenius.io/top-10-software-engineering-podcasts-in-2020-ce8171c47527?source=collection_archive---------0-----------------------#2020-05-08

听播客是我的爱好之一，我每天都花几个小时听。它已经成为我获取技术世界中新思想、技术、趋势和良好实践信息的来源之一。

我的一个主要问题是选择听哪一个，因为现在有这么多好的播客。有时很难集中注意力并选择一个。

我想很多想开始听播客的开发者都面临着同样的问题。所以我决定分享我最喜欢的 10 大软件工程播客。

编码块。Net: 这是我最喜欢的播客❤️.主持人讨论了软件工程领域的各种话题，并分享了他们的一些经验。他们有一些系列，包括选择一本书，每章都有一集。这是学习新概念和新技能的绝佳来源。目前，他们正在讨论《设计数据密集型应用》这本书。

链接:https://www.codingblocks.net/

软件工程广播:这是一个已经持续了一段时间的播客。他们每月发布 3 到 4 集新节目。每集都包括对软件工程领域专家的采访。如果你想了解软件工程的核心原则，这是一个很好的播客。

链接:https://www.se-radio.net/

干净的编码员:这是一个新的播客，关注如何开发高质量的软件。有一些关于代码质量、软件工程原则和敏捷开发的精彩讨论。我觉得这很有帮助，因为他们给出了一些你可以在工作中应用的好建议和想法。

链接:https://devchat.tv/clean-coders/

软技能:这是一个每周一次的播客，关注工程师的软技能。主持人分享他们的经验，并回答关于不同主题的电子邮件，例如晋升、技术领导力、辞职、冲突解决等等。它很有趣，因为它谈论了你通常在工作中面临的日常生活。

链接:https://softskills.audio/

软件工程日报:这是一个每日播客，有有趣的嘉宾和各种各样的话题。本播客展示了公司如何应对复杂性，以及他们应用了何种技术。我喜欢这个，因为它很实用。他们采访了中小型公司的员工，以展示如何在不同的规模上解决问题。

链接:【https://softwareengineeringdaily.com/ T4】

这是一个每周一次的播客，让你了解科技世界的最新动态。这条新闻以信息丰富和随意的方式呈现。这是很好的娱乐。

链接:https://www.theverge.com/the-vergecast

核心内容:这是一个针对技术话题的深度访谈播客。他们采访了许多有趣的人和各种主题，重点是函数式编程。

链接:https://corecursive.com/

变更日志:这是一个每周播客，包含来自技术世界的新闻和对软件社区中有影响力的人的采访。

链接:https://changelog.com/podcast

JavaScript Jabber: 这是一个专注于 JavaScript 的播客，已经持续了一段时间。他们每周都有一集。这几集是关于 JavaScript 世界的各种话题。包括编程实践、趋势、框架、编码环境以及社区的其他相关主题。

链接:https://devchat.tv/js-jabber/

被捕 DevOps: 这是一个关注 DevOps 的每周播客。他们就各种主题进行了有趣的讨论和采访，重点是帮助开发人员实现 DevOps 文化。

链接:https://www.arresteddevops.com/

很难从这么多优秀的播客中选出 10 个，但是我希望你在上下班的路上、在健身房或者在打扫房间的时候听得开心。享受并坚持学习🎧
