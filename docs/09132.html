帮助我成为 Laravel 网站开发人员的书籍

原文：https://blog.devgenius.io/books-that-helped-me-become-a-laravel-web-developer-2b8f855f94e1?source=collection_archive---------6-----------------------#2022-08-01

当 YouTube 教程还不那么流行的时候，我成为了一名网络开发人员。每个人都刚刚开始，我不知道从哪里开始。由于我对开发了解不多，我甚至不知道该问哪些问题。我通过从软件开发部门的一些同事那里听到的技术谈话，拼凑出了对 web 开发人员工作的初步理解(当时我还不在那个部门)。

我听到的是，“什么什么 PHP。”嗯，也许 PHP 可以帮我，所以我开始学习 PHP。我有足够的意识知道 HTML 是必需的。这是在 CSS 出现之前(想想表格)。非常幸运的是，我被一家公司雇佣了，这家公司想给他们的团队增加一名网络开发人员。剩下的就是历史了。

在我努力成为一名网络开发人员的过程中，以下几本书帮助了我。只是叫出来，我在开始看这些书之前是零编程知识，是它们救了我。

Head First PHP & MySQL:一个对大脑友好的指南

Head First PHP & MySQL:一个关于 Amazon.com 的大脑友好指南。*免费*送货到…

amzn.to

这本书我至少读过 10 遍了。即使现在回过头来看，我还是会嘲笑我曾经为之奋斗的东西，这是一本具有挑战性的书，但却是一本我能消化的书。当谈到 PHP 时,《PHP 和 MySQL》可能是最容易理解的书籍之一。这本书可能过时了，但仍然是一本好书。有趣的是，我正在写一本关于 PHP 的新书，应该会在今年年底出版。

Head First HTML 和 CSS:创建基于标准的网页的学习指南

厌倦了阅读只有成为专家后才有意义的 HTML 书籍？那么是时候你先捡起头了…

amzn.to

我知道你在说什么。HTML 和 CSS 不是编程语言。我仍然需要巩固这些概念。所以我先从头部开始。在那之前，是 W3Schools 。

Head First 编程:使用 Python 语言编程的学习指南

Head First 编程:使用 Python 语言编程的学习者指南[Griffiths，David，Barry，Paul]

amzn.to

我需要知道哪种编程语言适合我。所以我买了，你猜对了，另一本 Head First 的书。这是第一本专注于 Python 的编程书籍。混合语言可能是一个糟糕的举动，但它帮助我掌握了一些编程原则。

head First SQL:SQL 上的大脑-学习者指南

Head First SQL:你的大脑在 SQL 上——学习者指南[Beighley，Lynn]在 Amazon.com。*符合条件的免费*运输…

amzn.to

我觉得我需要更多的 SQL 经验，所以我又买了一本 Head First 的书。如果成功了，就成功了。我理解他们的概念。当时读我的 Java 书太有挑战性了。

用 Java 进行数据抽象和问题解决:墙和镜子

数据抽象和用 Java 解决问题:亚马逊网站上的墙和镜子

amzn.to

但是我觉得我仍然需要学习一门更严格的、面向对象的编程语言。所以我选了 Java。这是一本很难理解的书，但我还是勉强读完了，在网上搜索我无法理解的话题。

Sams 集 Ajax、Javascript 和 Php 于一身

针对 Web 开发人员和程序员的 Ajax、JavaScript 和 PHP 综合指南利用了一步一步的教程…

amzn.to

我读得越多，就越意识到我需要学习 JavaScript 和 Ajax。又花了几个月的时间让我的头脑重新连线来理解 Ajax，但是这本书确实帮助了我。

高级 PHP 编程

高级 PHP 编程:9780672325618:计算机科学书籍@ Amazon.com

amzn.to

我想认真学习 PHP，所以我购买了高级 PHP 编程。这本书，虽然已经开始显得过时了，但对我理解 PHP 有很大的帮助。

使用 PHP 和 MySQL 轻松实现电子商务(重要的声音)

在 Amazon.com 用 PHP 和 MySQL 轻松实现电子商务。*免费*送货到…

几年后，当我开始涉足电子商务领域时，我用 PHP 和 MySQL 购买了轻松电子商务。因为我已经足够精通 PHP/MySQL，所以这本书读起来很轻松。绝对是我的最爱之一。

动态网站的 PHP 和 MySQL:Visual quick pro 指南

这是我在开始阅读 Laravel 书籍之前阅读的最后一本关于纯 PHP 的书籍。容易消费，我相信我不到一周就完成了。

Laravel: Up & Running:构建现代 PHP 应用程序的框架

Laravel: Up & Running:在 Amazon.com 上构建现代 PHP 应用的框架。*免费*送货到…

这是 Laravel 的最新版本，我也读过，但让我进入 Laravel 的是第一版。我已经在用 Laravel 编写代码了，但是这些书帮助我坚定了用 Laravel 编写代码的决心。另一个有趣的事实是，我也在写一本拉勒维尔的书。
