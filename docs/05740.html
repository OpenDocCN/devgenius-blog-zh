5.迪奥

这个库支持 Android、IOS、Linus、MACOS、Windows 和 Web。 Dio 帮助你快速处理 HTTP。例如，你可以有一个 cookie 管理器，拦截器，或者通过这个库来处理错误或许多功能。而且超级好用；您可以如下实现它。

import 'package:dio/dio.dart';
void getHttp() async {
  try {
    var response = await Dio().get('http://www.google.com');
    print(response);
  } catch (e) {
    print(e);
  }
}
