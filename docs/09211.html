发生这种情况是因为我们处于测试环境中，包装 Android UI 线程的主调度程序将不可用，因为这些测试是在本地 JVM 而不是 Android 设备上执行的。因此，在启动协程之前，我们需要将 Dispatcher 设置为 Test Dispatcher。

Dispatchers.setMain(UnconfinedTestDispatcher())
