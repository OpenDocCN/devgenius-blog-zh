我们为什么要标准化数据？这个问题在我们脑海里蹦出来对吧？！井数据通常是非常混乱和肮脏的，有空值、重复、空白和大量冗余数据。

现在我们来看看重复和冗余之间的区别。

答案很简单，你把数据标准化。
