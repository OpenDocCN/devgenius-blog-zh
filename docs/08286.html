我们还将 HTTP 请求的属性和参数设置为“any”。有时开发人员使用这种方法来避免 Typescript 抛出错误。例如，当我们这样做的时候

如果我们不将响应声明为“any”，Typescript 会说类似于“是的，这是一个通用对象类型，我看不到里面的数据属性，因为它不是通用对象类型的一部分”。
