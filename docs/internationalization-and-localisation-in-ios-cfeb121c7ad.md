# 国际化和本地化的区别

**国际化**是指设计一个软件应用程序的过程，使其可以适应各种语言和地区，而无需进行工程更改。**本地化**是通过翻译文本和添加特定于地区的组件，使**国际化**软件适应特定地区或语言的过程。
