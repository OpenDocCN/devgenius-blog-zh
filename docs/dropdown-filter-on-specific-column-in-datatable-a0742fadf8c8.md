# 数据表中特定列的下拉筛选器

> 原文：<https://blog.devgenius.io/dropdown-filter-on-specific-column-in-datatable-a0742fadf8c8?source=collection_archive---------15----------------------->

在本文中，我们将了解如何在 datatable 中添加多个过滤器下拉列表。这个例子几乎与基于文本的单个列的例子相同，并提供相同的功能，但是在这个例子中，使用了`select`输入控件。

让我们看看顶部的 jquery 数据表列筛选器下拉列表、数据表搜索筛选器、数据表搜索多列、数据表特定列筛选器、数据表中特定列的下拉筛选器、数据表中的单个列搜索、特定列的 jquery 数据表下拉列表筛选、laravel 8/9 数据表特定列搜索、laravel 8 和 laravel 9 中的单个列搜索数据表

API 用于通过使用`[columns().every()](https://datatables.net/reference/api/columns().every())`循环遍历列来构建`select`输入(如果需要，选择器`[columns()](https://datatables.net/reference/api/columns())`也可用于限制所选列)，然后使用`[column().data()](https://datatables.net/reference/api/column().data())`方法依次获取每列的数据。

**Javascript:**

```
$(document).ready(function () {
    $('#example').DataTable({
        initComplete: function () {
            this.api()
                .columns()
                .every(function () {
                    var column = this;
                    var select = $('<select><option value=""></option></select>')
                        .appendTo($(column.footer()).empty())
                        .on('change', function () {
                            var val = $.fn.dataTable.util.escapeRegex($(this).val());

                            column.search(val ? '^' + val + '$' : '', true, false).draw();
                        });

                    column
                        .data()
                        .unique()
                        .sort()
                        .each(function (d, j) {
                            select.append('<option value="' + d + '">' + d + '</option>');
                        });
                });
        },
    });
});
```
