在今天的挑战中，我们将看到一种叫做MinStack的特殊“堆栈”类。

最后几行代码是我们需要分析的:首先，代码将一串数字推送到堆栈中，最后，它会询问答案:

obj = MinStack()
obj.push(...)
A = obj.top() + obj.min()
// A = ?

MinStack的构造函数创建了两个空数组MS和stack，这就是全部内容。push函数将参数添加到stack数组中，如果数字满足if条件，它也将被添加到MS。这个特定条件检查MS是否为空，或者MS中的最新元素是否大于或等于数字。简而言之，如果这个数字比里面的最新数字小(当然，如果它是空的),那么这个数字就会加到MS上。因此得名MinStack，它记录堆栈结构中最小的数字。
