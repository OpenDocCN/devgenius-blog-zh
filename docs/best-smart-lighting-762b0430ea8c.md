# 智能灯泡是如何工作的？

![](img/d37d7723aeb57ee034c47f6868348144.png)

智能灯泡使用无线技术连接到智能手机，就像最好的智能家居设备一样。它允许你从一个应用程序控制灯光。许多灯泡使用蓝牙连接到您的智能手机。但是当你不在或在不同的房间时，这些类型的灯是无法控制的。

如果你想在外出时控制你的照明系统，你可以寻找像飞利浦 Hue 这样有集线器的灯。
