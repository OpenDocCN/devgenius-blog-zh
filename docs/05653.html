如何为自己的下一次单口相声争取一个好的历史？

用 单口相声 。 从某些开发者那里寻找提交。您可以按年龄和姓名进行筛选。

要查找昨天的提交，请使用git standup。这将返回您最近提交的内容——非常适合每日站立。更多信息请使用git help standup。

如何把PR拉到你当地？

对于GitLab使用 git mr ，对于GitHub使用 git pr 。 这对代码评审来说很棒。此命令不适用于BitBucket。更容易地获取PR、检查变更和审查代码。

如何使用命令？ 运行git pr <PR-Number>

如何给.gitignore添加一个文件？

如何重置最后N次提交？

使用 git撤消 。 撤销上一次提交。您不需要重置为以前的。你根本不需要重置。使用git undo 4重置最后4次提交。

如何删除文件和相关提交？

可以用git obliterate。这将从存储库和相关提交中删除该文件。

如何修复特定的提交？

你知道如何修改提交。这将改变存储库中的最后一次提交。

您可以使用git commit --fixup <SHA>来修复特定的提交。然后使用git rebase -i --autosquash <SHA>来压缩这些修正。

这里有一个别名来加快速度。

如何找到修改最多的文件？

用 git努力 。 该命令给出了文件变动的良好度量。活动天数将显示修改花费了多少时间。

大部分提到的命令都需要 git-extras 。Git extras使Git变得更好、更少威胁、更快。
