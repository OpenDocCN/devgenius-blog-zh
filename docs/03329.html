第 43 天—生成二分搜索法树🏁

目的

给定一个整数n，生成所有存储值 1 的结构唯一的 BST 的(二分搜索法树)... n 。

Example🕶

Input: 3
Output:
[
  [1,null,3,2],
  [3,2,null,1],
  [3,1,null,null,2],
  [2,1,3],
  [1,null,2,null,3]
]
Explanation:
The above output corresponds to the 5 unique BST's shown below:

   1         3     3      2      1
    \       /     /      / \      \
     3     2     1      1   3      2
    /     /       \                 \
   2     1         2                 3

关注 代码之家 s 了解编程面试世界的最新动态。

密码👇

边缘案例

  1. 当我们使用递归函数来解决问题时，一定要记住添加基本情况。
  2. 这里的基本情况是，如果起始值大于结束值，那么我们必须退出递归函数

算法

  1. 对于每个索引值，将数组分成两个列表。
  2. 当前元素之前的元素将在左子树中，大于当前元素的元素将在右子树中。
  3. 针对左侧列表中的每个元素遍历右侧列表。
  4. 添加当前元素作为根值。左列表元素是左节点，右列表元素是右节点。
  5. 将树添加到结果列表中。
  6. 在遍历结束时返回结果列表。🔚

复杂性分析

时间复杂度:O(N)对于左列表中的每个元素，你遍历整个右列表。
空间复杂度:O(N)在两个列表中存储整个范围的元素
