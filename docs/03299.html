为了让您了解可用的选项，目前web开发中最常用的5种语言(排名不分先后)是:

  • 服务器端编程语言（Professional Hypertext Preprocessor的缩写）
  • 红宝石
  • 计算机编程语言
  • Java 语言(一种计算机语言，尤用于创建网站)
  • Java Script语言

你可能会看到一些人提到ASP.NET，这不是一种语言。这是一个框架。

我稍后会谈到区别，但现在我们只能说，虽然ASP.NET作为一个框架是一个有效的选择，但作为一个初学者，我不会担心它。

这可能会比你要求的多一点，但最终我认为它会对你的决定有更多的帮助。如果你已经知道下面的部分，请随意跳过。

web应用程序如何

你，用户，在你的 浏览器 的地址栏中输入一个 url 。

使用该url中的信息，您的浏览器识别您想要的内容以及在哪里查找它，并向应用程序的 服务器 发送 请求 。

这个请求是使用一个名为 HTTP 的 协议 发送的。一个协议仅仅意味着你的浏览器和服务器在相互通信时遵循一套规则。HTTP是一组特殊的规则。

当服务器收到浏览器发送的请求时，它会查看请求并决定将什么发送回浏览器。

当你第一次访问一个web应用时，这几乎总是一个 HTML 文档。

于是，服务器向你的浏览器返回一个 响应 ，其中包含HTML。

HTML包含浏览器向用户显示内容的指示。它还告诉您的浏览器向用户正确显示应用程序所需的其他内容。

主要是， CSS 和 JavaScript 文档，还有图片、字体等很多东西。

浏览器然后发出更多的请求来获取这些其他的东西，我们姑且称之为 资产 。

这些对资产的请求可以发送到任意数量的不同服务器，并且需要时间！

一个好的web应用程序会尽最大努力使这个过程高效且不被用户察觉。

网页中的一个 链接 就像是在你的地址栏中输入网址的快捷方式，很多时候当你输入一个网址或者点击一个链接的时候，这整个过程又重新开始了。

后端开发

后端开发都是关于服务器的。或者服务器。

因为web应用程序可能包含在许多不同的服务器上，或者是全部，或者是部分。

它是关于应用程序的服务器如何决定响应请求。

关键是他们的响应速度，以及他们可以同时响应多少个请求。

顺便说一下，对于web开发的初学者来说，前端开发几乎总是更重要。

即使你不关心让你的应用程序漂亮。

即使是你提到的第一个项目。我强烈建议你从专注于HTML、CSS和JavaScript开始。

web开发应该学习什么编程语言？

在我回答这个问题之前，我会给你一个框架的定义。

框架是运行应用程序的应用程序。

它为您提供了一套工具，使构建web应用程序变得更加容易。

框架的确切定义取决于你与谁交谈，但他们是为了让你的生活更轻松。

学习语言，然后学习你选择的框架。使用Google很容易找到任何语言或框架的指南和教程。

对于后端初学者，我既不推荐Java，也不推荐Javascript。

Java是一种比其他四种语言更复杂的语言，虽然说node.js(用于服务器的JavaScript)利用了您在学习前端开发时开发的技能很有诱惑力，但实际上我认为node.js比上面三种选项中的任何一种都不太适合初学者。
