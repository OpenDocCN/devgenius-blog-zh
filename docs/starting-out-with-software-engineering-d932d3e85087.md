# 软件和数字化转型

随着越来越多的组织进行数字化转型，熟练和合格的软件工程师很难找到，而且非常受欢迎。

数字化转型是将数字技术整合到企业的所有领域。各公司都在寻找方法来实现运营现代化，更好地利用技术。**数字化转型比其他任何事情都更有利于这一彻底改革。**

你甚至可以说，软件开发加速了公司进行的数字化转型的有效性，例如帮助自动化流程，改善沟通和协作，以及使数据更容易访问和使用。

就是这样。

# 软件职业

软件工程师可以在所有行业工作(汽车、能源、金融、零售、保险、制药等)。)、大大小小的软件公司、基于服务和产品的公司、政府以及私人组织甚至 R&D、研究机构和大学。

软件工程师选择专攻一个领域。许多职业道路摆在面前，确保可持续和体面的高薪工作:

*   后端开发人员
*   嵌入式软件开发人员
*   前端开发人员
*   移动开发者
*   大数据开发者
*   游戏开发商
*   定量显影剂
*   测试中的软件开发工程师
*   全栈工程师
*   图形工程师
*   DevOps 工程师
*   数据工程师
*   云工程师
*   研究工程师/研究软件开发工程师

# 资源👇

请按照列出的顺序浏览这些资源。如果有必要的话，把它们加入书签。

从今天开始是时候做实际工作了！

[哈佛大学 CS50【入门认证课程】](https://cs50.harvard.edu/)从这里开始。我认为这是对编程(通过 C)的最好介绍。这是网上最受欢迎的在线课程之一，甚至是免费的。导师大卫在这门课上投入了大量的精力和艺术。

[构建 Java 程序](https://www.amazon.com/-/de/dp/013547194X/ref=sr_1_1?__mk_de_DE=%C3%85M%C3%85%C5%BD%C3%95%C3%91&dchild=1&keywords=building+java+programs&qid=1634032853&sr=8-1) Java 是一种面向对象、安全、健壮的编译型编程语言。我推荐学习 Java 作为你的第一语言，因为我发现它最容易学。这本书教你足够的语言知识，让你开始学习。

[Java 中的数据结构&算法](https://www.amazon.com/-/de/dp/0672324539/ref=sr_1_1?__mk_de_DE=%C3%85M%C3%85%C5%BD%C3%95%C3%91&dchild=1&keywords=robert+lafore&qid=1634031320&sr=8-1)既然你已经理解了语言的语法(即语法和词汇)，是时候学习数据结构和算法(简称:DSA)了，它是高效代码的组成部分。这是我读过的最好的书之一。它包含了 Java 中每个基本 DSA 主题的清晰实现。

[甲骨文认证助理【初级认证】](https://education.oracle.com/oracle-certified-associate-java-se-8-programmer/trackp_333)这是甲骨文官方认证。甲骨文是一家拥有 Java 的技术公司。我推荐获得证书，因为我开始在求职中收到更多的回复。为了做好准备，我建议在阅读完[马拉·古普塔的书](https://www.amazon.com/-/de/dp/1617293253/ref=sr_1_11?__mk_de_DE=%C3%85M%C3%85%C5%BD%C3%95%C3%91&crid=GOM6ZN4B6653&dchild=1&keywords=oracle+certified+associate%2C+java+se+8+programmer&qid=1634220274&sprefix=oracle+cer%2Caps%2C240&sr=8-11)和[官方学习指南](https://www.amazon.com/OCA-Certified-Associate-Programmer-1Z0-808/dp/1118957407/ref=sr_1_2?crid=24ABFAGQG8WAK&dchild=1&keywords=oracle+certified+associate%2C+java+se+8+programmer&qid=1634752510&qsid=136-6622026-3252607&sprefix=java+certified+ass%2Caps%2C364&sr=8-2&sres=1260011399%2C1118957407%2C1976704030%2C0132930218%2C1534931023%2CB00TVERES2%2C1259587517%2C0789759241%2C1119272092%2CB08DF4R2V9%2CB097826DWQ%2C1484218353%2CB019NGYH8W%2C1119696135%2C1260440230%2CB01A1MN0ZS&srpt=ABIS_BOOK)后，做[模拟考试](https://www.udemy.com/course/oracle-java-associate-certification-exam-course-1z0-808/)。

[SQL 快速入门指南](https://www.amazon.com/-/de/dp/B081P1YYFB/?_encoding=UTF8&pd_rd_w=1T9iR&pf_rd_p=6e9da02f-f7a3-444f-aea6-9ef09ed8bb89&pf_rd_r=J2KY0XENGFQCDRSNTC8K&pd_rd_r=1aaafd3d-e98f-4713-8517-fa692f501a56&pd_rd_wg=128ET&ref_=pd_gw_ci_mcx_mr_hp_d)我非常喜欢这本书！在大学研究玩具数据集只能让你到此为止。处理真实世界的数据要复杂得多，只有更深入地理解它才能带来真正的商业价值。

[用 Python 自动完成枯燥的工作](https://www.amazon.com/-/de/dp/1593279922/ref=sr_1_1?__mk_de_DE=%C3%85M%C3%85%C5%BD%C3%95%C3%91&dchild=1&keywords=automate+python&qid=1634041870&sr=8-1)工具箱中的两种语言总比一种好！学习一种脚本语言，即代码不是首先被翻译成机器代码而是立即运行的编程语言，如 Python 或 Ruby，会使你在自动化事物和编写辅助程序方面更有效率。

[HackerRank【用于面试准备】](https://www.hackerrank.com/) HackerRank 是一个免费的编码挑战网站，有 100 多个不同难度的问题。有超过 1500 万。HackerRank 上的用户。他们的内置编辑器允许你用自己选择的语言编程。他们的问题类似于真实的面试问题。偶尔也有比赛。
