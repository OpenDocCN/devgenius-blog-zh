使用Quasar图书馆卡开发Vue应用程序

原文：https://blog.devgenius.io/developing-vue-apps-with-the-quasar-library-cards-784933687a81?source=collection_archive---------6-----------------------#2021-01-22

Quasar是一个流行的Vue UI库，用于开发好看的Vue应用程序。

在本文中，我们将了解如何使用Quasar UI库创建Vue应用程序。

卡片

q-card组件让我们添加一个内容容器来显示分组的内容。

例如，我们可以写:

<!DOCTYPE html>
<html>
  <head>
    <link
      href="https://fonts.googleapis.com/css?family=Roboto:100,300,400,500,700,900|Material+Icons"
      rel="stylesheet"
      type="text/css"
    />
    <link
      href="https://cdn.jsdelivr.net/npm/quasar@1.12.13/dist/quasar.min.css"
      rel="stylesheet"
      type="text/css"
    />
  </head>
  <body class="body--dark">
    <script src="https://cdn.jsdelivr.net/npm/vue@^2.0.0/dist/vue.min.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/quasar@1.12.13/dist/quasar.umd.min.js"></script>
    <div id="q-app">
      <q-layout
        view="lHh Lpr lFf"
        container
        style="height: 100vh;"
        class="shadow-2 rounded-borders"
      >
        <q-card
          class="my-card text-white"
          style="
            background: radial-gradient(circle, lightblue 0%, #014a88 100%);
          "
        >
          <q-card-section>
            <div class="text-h6">The Planet</div>
            <div class="text-subtitle2">by John Doe</div>
          </q-card-section>          <q-card-section class="q-pt-none">
            Lorem ipsum
          </q-card-section>
        </q-card>
      </q-layout>
    </div>
    <script>
      new Vue({
        el: "#q-app",
        data: {}
      });
    </script>
  </body>
</html>

添加不同部分的卡片。

q-card-section让我们将卡片分成几部分。

卡片操作

我们可以用q-card-actions组件来实现acc卡的动作按钮。

例如，我们可以写:

<!DOCTYPE html>
<html>
  <head>
    <link
      href="https://fonts.googleapis.com/css?family=Roboto:100,300,400,500,700,900|Material+Icons"
      rel="stylesheet"
      type="text/css"
    />
    <link
      href="https://cdn.jsdelivr.net/npm/quasar@1.12.13/dist/quasar.min.css"
      rel="stylesheet"
      type="text/css"
    />
  </head>
  <body class="body--dark">
    <script src="https://cdn.jsdelivr.net/npm/vue@^2.0.0/dist/vue.min.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/quasar@1.12.13/dist/quasar.umd.min.js"></script>
    <div id="q-app">
      <q-layout
        view="lHh Lpr lFf"
        container
        style="height: 100vh;"
        class="shadow-2 rounded-borders"
      >
        <q-card class="my-card">
          <q-card-section class="bg-primary text-white">
            <div class="text-h6">The Planet</div>
            <div class="text-subtitle2">by John Doe</div>
          </q-card-section>          <q-separator></q-separator>          <q-card-actions align="right">
            <q-btn flat>Action 1</q-btn>
            <q-btn flat>Action 2</q-btn>
          </q-card-actions>
        </q-card>
      </q-layout>
    </div>
    <script>
      new Vue({
        el: "#q-app",
        data: {}
      });
    </script>
  </body>
</html>

我们添加q-card-actions组件将按钮添加到底部。

align设置为right使按钮显示在卡片的右侧。
