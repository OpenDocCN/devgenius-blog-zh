GitHub 初学者指南。

什么是 GitHub？

GitHub 是一个 Git 存储库版本控制系统，您可以在终端工作时使用。版本控制允许任何人在项目上同时协作，对文件进行修改，并保存或“提交”。Git 跟踪对文件所做的更改，这样您就有了所做工作的记录，并且如果需要的话，您可以恢复到文件的以前版本。
