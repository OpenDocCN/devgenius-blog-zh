让我们深入一点。

  • 问题 Seaborn 是如何更新 matplotlib 数据的？
  • 回答— Seaborn 更新 matplotlib rc 参数。术语“ rc ”代表运行命令。当您创建新图形时，这些默认设置将被初始化。因此，Seaborn 修改了的一些默认设置，比如的背景颜色、网格颜色、字体大小等。你可能听说 Seaborn 是 matplotlib 的包装器，这仅仅意味着 Seaborn构建在 matplotlib 代码之上，但是给了你额外的功能，更好的语法，更好的默认值。然而，它从 matplotlib 继承了许多代码，所以您可以继续使用许多 matplotlib 代码来更新您的 Seabourn 图形。
