准备…设置…开始

假设您的开发机器安装了Gradle(在撰写本文时，我使用的是Gradle 5.3.1

Create a folder 'helloworld'

在文件夹下创建以下三个与Gradle相关的文件

Build.gradle
gradle. properties
settings.gradle

并在gradle命令下运行以创建Gradle包装器

gradle wrapper --gradle-version 5.6.2
