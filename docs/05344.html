DNS，域名服务器？

域名和域名服务器是密切相关的。

为了让我们理解什么是域名服务器，我们需要知道什么是域名。

什么是域名？

简单地说，域名就是你网站的 URL 名称。它取代了使用网站的 IP 地址。例如，我们可以使用脸书的 IP 地址访问他们的网站(FB: 69.63.181.15)。然而，谁会记得呢？所以他们注册了一个域名供用户访问网站。(www.facebook.com)。

什么是域名服务器？

DNS 保存网站名称(DN)的名称，当用户请求网站时，DNS 将网站名称(DN)与正确的 IP 地址进行匹配。一旦服务器找到它，服务器就发回该网站的所有内容。当然，这是幕后发生的事情。
