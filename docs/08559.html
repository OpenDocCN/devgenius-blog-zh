什么是 JavaFX

JavaFX 是一个软件平台，旨在创建和运行桌面、web 和移动应用程序。以下是它历史的一瞥:

  • 2008 年—首次发布
  • 2011 年——甲骨文开始开源这个项目，作为 OpenJDK 的一部分，名为 OpenJFX
  • 未来——将继续由胶子公司支持

编译和运行 JavaFX 三种方法

  1. 基本项目
  2. 使用 Maven
  3. 使用 Gradle

您也可以查看 OpenJFX 网站以获取更多补充信息。
