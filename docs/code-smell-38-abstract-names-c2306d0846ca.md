# 问题

*   断开[映射器](https://codeburst.io/what-is-software-9a78c1172cf9)和[到现实世界实体的双投影](https://codeburst.io/the-one-and-only-software-design-principle-5328420712af)。

# 解决方法

1.  选择有意义的名字。
2.  找隐喻。
3.  避免使用*抽象*、*基础*、*类属*、*帮助者*等词语。
4.  使用[规则](https://medium.com/dev-genius/what-exactly-is-a-name-part-ii-rehab-fc5058b3ff1)进行命名。
