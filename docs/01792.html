我们来简单分析一下这几行。对象matrix被用作 2D 数组(就像一个由行和列组成的网格)。

第一个 if 条件检查第一列中的所有元素是否都等于🍎
:matrix[0][j] == 🍎

第二个条件检查💰第-行等于零:
matrix[i][💰] == 0
