函数可以返回值:我说过了。当我们之前看函数的时候，那些函数只是重复一些字符串。尽管这是函数的一种用途，但函数比简单的回显 out语句更强大。我们可以使用函数做一些事情，然后返回一个计算出来的值给我们。然后，该值可以被回显或赋给一个变量。

要返回一个值，您将在函数内部使用 return 关键字。确保 return 语句是函数中的最后一条语句，因为PHP将在执行 return 语句后退出函数。

  1. PHP通过调用 do_secret_calculation(10，33) 函数并传递给它两个参数 10 和 33 ，在第13行开始执行过程。

让我们重温一下我之前提到的一个语句:PHP一完成 return 语句就退出函数。

这段代码是怎么回事？

  1. 第11行调用了函数 assign_one() 。
  2. PHP进入函数 assign_one() ，将值1赋给一个局部变量 $x 。
  3. 然后转到下一条语句，并返回存储在 $x 中的值。
  4. PHP退出函数并返回到第11行。它不而执行第8行的 $x++ 。
  5. 值1被分配给变量 $one 。

如你所见，PHP永远不会到达 return 语句之后的语句。

函数可以返回任何数据类型:它们不需要返回整数值。您也不必像我们到目前为止所做的那样，将函数调用赋给一个变量或在函数内部使用一个局部变量(我这样做只是为了明确)。

在下面的例子中，函数 display_two() 返回字符串 two 。我们可以在 echo 语句中连接函数调用。这样，PHP看到了 echo 语句并开始执行。它尝试将两个值合并在一起，因为它遇到了串联运算符。在这样做之前，它首先必须调用函数 display_two() ，该函数返回值 two。两个字符串被连接，完整的字符串被回显出来。
