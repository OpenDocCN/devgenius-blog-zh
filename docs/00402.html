介绍

在本教程中，我们将看看如何用Spring WebFlux创建一个基本的websocket连接。

WebSocket是web浏览器和服务器之间通过单一TCP连接进行双向通信的协议。因此，它在聊天应用程序或任何基于事件的通信中都很有用。换句话说，您可以在一个请求(连接)中流数据。

本页涵盖以下主题:

  3. 怎么发消息？
  4. 编写演示项目。
  5. 如何保护websockets？

如何保护websockets？

  1. 使用SSL/TLS上的web sockets(was://protocol)。
  2. 验证客户端和服务器数据。
  3. 仅在授权后使用websocket连接。
  4. 在websocket连接之前，生成可以通过HTTP检索的用户唯一推送id，并使用它与服务器通信。
  5. 使用原始标题。
