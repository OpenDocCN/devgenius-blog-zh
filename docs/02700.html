第31天——使用回溯进行回文分割

目的

给定一个字符串 s ，划分 s 使得划分的每个子串都是一个回文。

返回 s 的所有可能的回文划分。

例子

Input: "aab"
Output:
[
  ["aa","b"],
  ["a","a","b"]
]

关注 代码之家 s 了解编程面试世界的最新动态。

密码👇

算法👨‍🎓

  1. 你必须递归地使用DFS算法来回溯。
  2. 你从指向同一个字符的开始指针和结束指针开始。
  3. 对于开始和结束指针之间的每个字符组合，验证该字符串是否是回文。
  4. 如果是回文，将子串添加到路径列表中。当结束指针到达字符串的末尾时，通过改变起点来重复。
  5. 每当您到达字符串的末尾时，添加结果的路径。记住，只有当结果不为空时，才将列表添加到结果中。🔚
