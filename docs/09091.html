问题:→

给定两个二进制字符串a和b，将它们的和作为二进制字符串返回。

例 1: 

Input: a = "11", b = "1"
Output: "100"

例 2: 

Input: a = "1010", b = "1011"
Output: "10101"

约束:

  • 1 <= a.length, b.length <= 104
  • a和b仅由'0'或'1'字符组成。
  • 除了零本身，每个字符串不包含前导零。
