# 结论

计算机科学诞生于科学之母(数学)。在数学中，单字母变量的赋值 *(i，j，x，y)* 是一个很好的练习。

指称的概念产生于变量。

许多人想知道为什么数学家能处理如此短的变量，而计算机科学家却不能。

对于数学家来说，一旦进入公式，变量就失去了所有的语义，变得无法区分。

我们的大脑浪费了大量的精力去理解缩写的意思。

现在是 2020 年，我们需要为人类编写软件，而不是为编译器。
