# InOut 参数

在编程和使用泛型的过程中，我们经常会发现自己在使用 getter 和 setter 语句，同时也在使用输入参数。在编程中，我们不能改变函数中的参数。InOut 语句允许我们修改函数中的参数。这些变化都将发生在我们的身体内外。

> **&** :使用 Inout 参数时，我们在调用我们的函数时会注意到这个符号。所有的字符都作为变量在程序中的地址。
