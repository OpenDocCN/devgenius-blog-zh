问题:→

给定两个二叉树p和q的根，写一个函数检查它们是否相同。

如果两个二叉树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

例 1: 

Input: p = [1,2,3], q = [1,2,3]
Output: true

例 2: 

Input: p = [1,2], q = [1,null,2]
Output: false

例 3: 

Input: p = [1,2,1], q = [1,1,2]
Output: false

约束:

  • 两棵树中的节点数量都在[0, 100]范围内。
  • -104 <= Node.val <= 104
